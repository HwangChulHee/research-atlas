# 2604.17931 LiteResearcher: A Scalable Agentic RL Training Framework for Deep Research Agent

> section_tag: 검색시점;RAG강화추론  ·  survey_source: s3

## 초록 (한글)
강화학습(RL)은 LLM 기반 에이전트의 강력한 학습 패러다임으로 부상했다. 그러나 딥 리서치를 위한 에이전트형 RL 확장은 두 가지 결합된 난제로 제약된다. 수작업 합성 데이터는 진정한 실세계 검색 능력을 끌어내지 못하고, RL 훈련 중 실세계 검색 의존은 불안정성과 막대한 비용을 유발해 에이전트형 RL의 확장성을 제한한다. LiteResearcher는 에이전트형 RL을 확장 가능하게 만드는 훈련 프레임워크로, 실세계 검색 동역학을 모방하는 경량 가상 세계를 구축해, 작은 검색 에이전트가 대규모 오픈소스·상용 모델(예: Tongyi DeepResearch, Claude-4.5 Sonnet)을 능가하는 지속적 개선 레시피를 가능케 한다. 구체적으로 GAIA와 Xbench 같은 공통 벤치마크에서 LiteResearcher-4B는 각각 71.3%, 78.0%의 오픈소스 SOTA를 달성해, 확장 가능한 RL 훈련이 딥 리서치 에이전트의 핵심 동력임을 입증한다.

## Introduction (한글)
추론 모델에 대한 RL의 최근 진전은 안정적 훈련 환경과 적절한 난이도 데이터 하에서 모델 능력의 지속적 확장이 가능함을 보였다. 그러나 추론을 도구 사용과 통합해야 하는 에이전트형 RL로 넘어가면, 기존 노력들은 유사한 지속적 개선 달성에 어려움을 겪었다. 주된 병목은 실세계 환경의 내재적 잡음·불안정성이다. 코딩 에이전트 영역에서는 생산 환경을 모사하면서 인프라 잡음을 걸러내는 격리 샌드박스로 이를 효과적으로 해결해 확장 가능한 훈련을 이뤘다.

그러나 딥 리서치 영역은 아직 이런 확장 가능 RL 훈련을 실현하지 못했다. 온라인 RL 프레임워크는 라이브 인터넷과 직접 상호작용해 큰 훈련 분산과 막대한 비용을 유발하고, 로컬 검색 기반 시스템은 좁은 코퍼스에 갇혀 다양한 인터넷의 복잡한 검색 동역학을 복제하지 못하며, 현재 실웹 그래프 기반 데이터 합성은 과도하게 설계된 추론 구조에 의존해 교차검증·열거 같은 원자적 검색 기법의 다양성을 무시한다.

저자들은 에이전트형 RL을 딥 리서치로 확장하려면 라이브 인터넷 구조를 모방하되 실행은 격리된 가상 세계가 필요하다고 주장한다—이 트윈 구조는 가상 환경에서 최적화된 정책이 오픈 웹으로 일반화되도록 하고, 격리는 환경 잡음의 높은 분산으로부터 훈련을 보호한다. 본 논문은 실세계 웹 구조를 모방하는 가상 세계를 구축하는 에이전트형 RL 훈련 프레임워크 LiteResearcher를 제시한다. (1) 실 검색 데이터 합성, (2) 검색 환경 시뮬레이션, (3) 커리큘럼 RL을 통한 지속 개선의 단계를 따른다. 정보원 규모를 확장하면 단순하지만 효과적인 데이터 합성으로도 현실적 훈련 과제 분포를 얻을 수 있음을 발견했다. 각 합성 과제마다 관련 실세계 웹페이지를 검색·수집해 로컬 코퍼스를 확장함으로써, 로컬 검색·브라우즈 도구가 오픈 웹과 유사한 검색 동역학을 제공하도록 한다. 또한 난이도 인지 커리큘럼 RL을 적용해 단계마다 과제 난이도와 맥락 길이를 점진 증가시키고, 현재 모델이 부분적으로 풀 수 있는 과제만 남겨 일관된 훈련 신호를 제공한다. LiteResearcher-4B는 GAIA 71.3%, Xbench 78.0%를 달성해 최대 8배 큰 오픈소스 에이전트(Tongyi DeepResearch 30B, WebSailor 30B)를 능가하고 상용 시스템(Claude-4.5-Sonnet, GLM-4.6)에 필적·초과한다.

## 한 줄 핵심
라이브 인터넷을 모방하되 격리 실행되는 "경량 가상 세계"와 난이도 커리큘럼 RL로, 작은 4B 딥 리서치 검색 에이전트를 안정적·저비용으로 SOTA까지 확장하는 훈련 프레임워크.
