# 2508.17188 PosterGen: Aesthetic-Aware Paper-to-Poster Generation via Multi-Agent LLMs

## 초록 (한글)
대형 언어 모델(LLM)과 멀티모달 대형 언어 모델(MLLM)에 기반한 멀티에이전트 시스템은 최근 복잡한 합성(compositional) 과제에서 강한 능력을 보였다. 본 논문은 이 패러다임을 논문-투-포스터 생성에 적용한다. 이는 연구 소통에서 실용적이지만 시간이 많이 드는 단계다. 기존 방법 대부분은 핵심 디자인·미학 원칙을 무시하여, 여전히 상당한 수작업 다듬기를 요하는 포스터를 산출한다. 이를 해결하기 위해, 근본적 디자인 원칙을 전문 디자이너를 본뜬 전용 에이전트 워크플로우에 내장한 멀티에이전트 프레임워크 PosterGen을 제안한다. 본 시스템은 협력 에이전트들을 포함한다: 논문의 서사를 추출하는 CuratorAgent, 콘텐츠를 균형 잡힌 공간 레이아웃으로 사상하는 LayoutAgent, 색상·타이포그래피의 일관된 시각 체계를 적용하는 StylistAgents, 최종 포스터를 PPTX 형식으로 렌더링하는 Renderer. 이 디자인 중심 접근은 의미적으로 근거 있고 시각적으로 매력적인 포스터를 만든다. 시각 품질을 체계적으로 평가하기 위해, 정보 충실도와 미적 디자인을 측정하는 포괄적 VLM 기반 루브릭을 도입하고 사용자 연구를 통해 인간 선호와 대조 검증한다. 실험 결과 PosterGen은 인간 디자인 포스터에 필적하는 콘텐츠 충실도와 디자인 미학을 달성하면서 최첨단 방법들을 일관되고 유의하게 능가한다.

## Introduction (한글)
학술 포스터는 학회에서 빠르고 시각적이며 상호작용적인 연구 소통의 필수 매체로, 청중과 저자 간 일대일 대화의 수단을 제공한다. 그러나 많은 연구자가 포스터의 신중한 디자인에 상당한 노력과 시간을 들여 고통받는다. 따라서 수작업 부담을 크게 덜어줄, 고품질 학술 포스터 자동 생성 방법이 절실하다.

상업용 포스터 자동 생성은 상당한 관심을 받아왔지만, 학술 포스터 생성 연구는 훨씬 덜 탐구되었다. 초기 학술 포스터 연구는 신경망 모델을 쓰거나 레이아웃 생성·텍스트 요약 같은 특정 하위 과제에 집중했다. 이들은 콘텐츠 오버플로우 같은 품질 문제를 낳고 추가 수작업 조정을 요하는 경우가 많다. 최근 LLM 기반 멀티에이전트 시스템이 복잡 과제에서 강한 성능을 보였다. P2P와 PosterAgent가 학술 포스터 생성에 이 접근을 처음 적용했으나, 미학과 디자인 원칙(자연스러운 읽기 흐름을 보장하는 잘 조직된 레이아웃, 시각적 위계를 위한 색·타이포 선택)을 충분히 고려하지 못했다.

현재 접근의 미적 한계를 넘고 수작업 다듬기 필요를 최소화하기 위해, 인간 수준 미적 완성도에 가까운 학술 포스터를 생성하도록 설계된 최초의 미학 주도 멀티에이전트 프레임워크 PosterGen을 제안한다. Table 1에서 보듯 기존 방법은 부분 해법만 제공하지만, PosterGen은 원리적 레이아웃과 스타일링을 모두 완전히 통합하면서 LaTeX·HTML보다 접근성 높은 PPTX 형식을 지원하는 최초의 방법이다. 이러한 미적 원칙 내장이 완전 자동화에 필수적이라고 주장한다.

PosterGen은 전문 디자인 과정을 본뜬 협력 멀티에이전트 워크플로우를 구현하며 네 핵심 단계로 구성된다: (i) 핵심 서사·시각 자산 증류, (ii) ABT 구조 기반 서사 주도 스토리보드 구성, (iii) 스토리보드를 균형·미학 인식 레이아웃으로 사상, (iv) 일관된 색·폰트 체계 적용. VLM-as-Judge와 인간 평가로 SOTA 방법들과 비교한 결과 주요 발견: 텍스트-투-이미지 방법(GPT-4o-Image)은 콘텐츠 환각·깨진 텍스트를 자주 일으키며, PosterGen은 인간 디자인 포스터에 필적하는 콘텐츠·미적 품질을 달성하면서 거의 모든 지표에서 SOTA 멀티에이전트 방법을 일관되게 능가하고, 디자인 중심 접근이 매우 효과적임을 양적·질적으로 확인한다.

## 한 줄 핵심
전문 디자이너 작업 흐름을 본뜬 멀티에이전트(콘텐츠·레이아웃·스타일·렌더)로 논문을 미학적으로 잘 디자인된 PPTX 포스터로 자동 변환하고, 이를 평가할 VLM 기반 루브릭까지 제안한 방법.
