# 2605.07177 HyperEyes: Dual-Grained Efficiency-Aware Reinforcement Learning for Parallel Multimodal Search Agents

> section_tag: RAG강화추론;평가  ·  survey_source: s3

## 초록 (한글)
기존 멀티모달 검색 에이전트는 대상 개체를 순차적으로 처리해 개체당 하나의 도구 호출을 발행하므로, 질의가 독립적 하위 검색으로 자연 분해될 때마다 중복 상호작용 라운드를 누적한다. 이런 분해 가능 질의에 대해, 효과적 멀티모달 에이전트는 순차적(longer)이 아니라 한 라운드 내에서 여러 근거 질의를 동시 발행하는 폭넓은(wider) 검색을 해야 한다고 주장한다. 이를 위해 시각 그라운딩과 검색을 단일 원자 동작으로 융합해 여러 개체를 동시 검색하면서 추론 효율을 일급 훈련 목표로 다루는 병렬 멀티모달 검색 에이전트 HyperEyes를 제시한다. HyperEyes는 두 단계로 훈련된다. 콜드스타트 지도를 위해 시각 다개체·텍스트 다제약 질의를 다루는 Parallel-Amenable Data Synthesis Pipeline을 개발하고 Progressive Rejection Sampling으로 효율 지향 궤적을 정제한다. 이를 바탕으로 핵심 기여인 Dual-Grained Efficiency-Aware RL 프레임워크는 두 보완 수준에서 작동한다. 거시 수준에서는 훈련 중 기준을 단조롭게 조이는 궤적 수준 보상 TRACE(Tool-use Reference-Adaptive Cost Efficiency)로 진정한 멀티홉 검색을 과도 제한하지 않으면서 불필요한 도구 호출을 억제한다. 미시 수준에서는 On-Policy Distillation(OPD)을 멀티모달 에이전트형 검색에 적응시켜 실패 롤아웃에 외부 교사의 밀집 토큰 수준 교정 신호를 주입해 희소 결과 보상의 신용 할당 결핍을 완화한다. 기존 멀티모달 검색 벤치마크 대부분이 정확도만 평가하고 추론 비용·병렬 검색 능력을 빠뜨리므로, 300개 다개체 시각 인스턴스로 정확도와 효율을 함께 평가하는 인간 정제 벤치마크 IMEB를 도입한다. 6개 벤치마크에서 HyperEyes-30B는 비슷한 규모의 최강 오픈소스 멀티모달 검색 에이전트 대비 정확도 9.9% 향상, 평균 도구 호출 라운드 5.3배 감소를 달성한다.

## Introduction (한글)
LLM과 MLLM의 파라미터 지식은 학습 데이터 컷오프로 구조적으로 제약되며, 이것이 외부 검색 도구를 적극 호출해 응답을 실시간·검증 가능 정보에 근거화하는 검색 에이전트의 개발을 이끈다. 그러나 멀티모달 검색 에이전트의 지배적 패러다임은 추론 사슬을 깊게 하기 위해 순차적 도구 호출에 크게 의존한다. 멀티홉 추론에는 효과적이지만, 질의가 독립적 하위 검색으로 분해될 수 있을 때 심각한 상호작용 중복을 유발한다.

병렬 도구 호출이 텍스트 기반 에이전트와 최근 시각 모델에서 이 병목을 다루려 등장했지만, 병렬 능력 보유가 효율적 검색 행동을 보장하지는 않는다. 기존 모델은 주로 순수 정확도 보상으로 최적화되어 장황한 궤적보다 간결한 병렬 궤적을 선호할 유인이 없다. 명시적 효율 목표가 없으면 병렬 능력은 무차별적 과잉 검색으로 퇴화해, 정확도 회복을 위해 불필요한 상호작용 라운드를 거치게 된다.

이 근본적 비효율 극복을 위해 "더 길게가 아니라 더 넓게 검색하라"는 원칙으로 HyperEyes를 제안한다. 시각 그라운딩과 검색을 단일 원자 동작으로 융합하는 Unified Grounded Search(UGS) 행동 공간에서 작동해 텍스트 수준 병렬성을 시각 모달리티로 확장한다. 학습된 정책이 병렬적이고 엄격히 비중복이 되도록, 효율을 일급 최적화 목표로 다루는 Dual-Grained Efficiency-Aware RL 프레임워크와 결합한다. 거시 수준에서 훈련 중 동적으로 조여 정책을 최적 효율로 안내하는 궤적 수준 기준 TRACE를, 미시 수준에서 실패 롤아웃에 전문가 교사의 밀집 토큰 수준 감독을 제공해 모호한 신용 할당을 해소하는 On-Policy Distillation(OPD)을 도입한다. 또한 Progressive Rejection Sampling으로 고품질·효율 지향 콜드스타트 궤적을 정제하는 Parallel-Amenable Data Synthesis Pipeline으로 이 훈련 패러다임을 뒷받침한다.

한편 표준 평가는 주로 최종 정답 정확도만 평가해 장황한 검색 궤적의 비효율을 가린다. 병렬 검색의 효율 이득을 정량화하기 위해, 멀티모달 검색 에이전트를 정확도와 검색 효율 양면에서 공동 평가하는 인간 정제 데이터셋 IMEB(Image Multi-Entity Benchmark)를 도입한다—각 인스턴스는 다개체 이미지와 다개체 동시 지역화·검색을 엄격히 요구하는 질문 쌍이다. 이 종합 평가에서 병렬 검색 폭이 다개체 시각 검색의 주된 병목임을 보인다. 기여 요약: (1) UGS 행동 공간의 효율적 병렬 멀티모달 검색 에이전트 HyperEyes, (2) 정확도와 검색 효율을 공동 평가하는 최초의 인간 정제 벤치마크 IMEB, (3) 6개 벤치마크에서 SOTA 및 최강 오픈소스 에이전트 대비 정확도 9.9% 향상·도구 호출 5.3배 감소.

## 한 줄 핵심
시각 그라운딩과 검색을 한 동작으로 융합해 여러 개체를 한 턴에 병렬 검색하고, 거시(TRACE)·미시(OPD) 이중 입도 효율 보상 RL로 과잉 검색을 억제하는 효율 중심 멀티모달 검색 에이전트.
