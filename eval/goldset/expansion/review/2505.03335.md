# 2505.03335 Absolute Zero: Reinforced Self-Play Reasoning with Zero Data

> section_tag: 학습형RL  ·  survey_source: s2

## 초록 (한글)
검증 가능 보상을 사용하는 강화학습(RLVR)은 규칙 기반 결과 보상으로부터 직접 학습함으로써 LLM의 추론 능력을 향상시킬 잠재력을 보여왔다. 최근 "zero" 설정에서 작동하는 RLVR 연구들은 추론 과정 라벨링의 감독을 피하지만, 여전히 수작업으로 큐레이션된 질문-답변 모음에 의존해 학습한다. 고품질 인간 생성 예시의 희소성은 인간 감독에 대한 장기적 확장성에 우려를 낳으며, 이는 LLM 사전학습 영역에서 이미 드러난 난제다. 또한 AI가 인간 지능을 능가하는 가상의 미래에는 인간이 제공한 과제가 초지능 시스템에 제한된 학습 잠재력만 줄 수 있다. 이를 해결하기 위해, 단일 모델이 자신의 학습 진전을 극대화하는 과제를 스스로 제안하고 그것을 풀며 추론을 개선하는, 외부 데이터에 전혀 의존하지 않는 새로운 RLVR 패러다임 "Absolute Zero"를 제안한다. 이 패러다임 하에서, 코드 실행기를 사용해 자기 제안 코드 추론 과제를 검증하고 답을 확인하는 통합된 검증 가능 피드백원으로 삼아 학습 커리큘럼과 추론 능력을 자기 진화시키는 시스템 Absolute Zero Reasoner(AZR)를 도입한다. 외부 데이터 없이 학습되었음에도 AZR은 코딩·수학 추론 과제에서 전반적 SOTA를 달성하며, 수만 개의 도메인 내 인간 큐레이션 예시에 의존하는 기존 "zero" 모델들을 능가한다. AZR은 다양한 모델 규모와 모델 클래스에 효과적으로 적용된다.

## Introduction (한글)
LLM은 RLVR을 통해 추론 능력에서 큰 향상을 이뤘다. 중간 추론 단계를 명시적으로 모방하는 방법과 달리, RLVR은 결과 기반 피드백만 사용해 방대한 과제 데이터셋에 대한 대규모 RL을 가능하게 한다. 특히 "zero" RLVR 패러다임은 cold-start 증류 데이터(인간이든 AI든 추론 궤적)를 전혀 쓰지 않고 베이스 모델에 직접 과제 보상으로 RLVR을 적용한다. 그러나 이런 방법들도 여전히 전문가가 큐레이션한 추론 질문-답변 쌍 분포에 크게 의존해, 장기적 확장성에 심각한 우려를 낳는다. 추론 모델이 발전할수록 대규모 고품질 데이터셋 구축 노력이 곧 지속 불가능해질 수 있다. 이는 LLM 사전학습에서 이미 확인된 확장성 병목과 유사하다. 또한 AI가 인간 지능을 넘어설 경우, 인간이 설계한 과제에만 의존하는 것은 자율 학습·성장 능력에 제약이 된다.

이에 "Absolute Zero" 패러다임을 제안한다. 모델이 학습 가능성을 극대화하는 과제를 정의하는 법과 그것을 효과적으로 푸는 법을 동시에 학습해, 외부 데이터 없이 자기 대국(self-play)을 통한 자기 진화를 가능케 한다. 좁은 도메인, 고정 기능, 또는 해킹에 취약한 학습된 보상 모델에 국한된 기존 자기 대국 방법과 달리, Absolute Zero 패러다임은 개방형 환경에서 작동하면서도 실제 환경에 근거를 둔다. 환경 피드백을 검증 가능한 보상원으로 삼아, 인간이 세계와의 상호작용으로 배우고 추론하는 방식을 모방하며, 신경망 보상 모델 해킹 같은 문제를 방지한다. 자기 대국으로 개선되는 AlphaZero와 마찬가지로, 본 패러다임은 인간 감독 없이 자기 상호작용만으로 학습한다.

이 패러다임 위에서 코드 추론 과제를 제안하고 푸는 Absolute Zero Reasoner(AZR)를 도입한다. 코드 실행기를 개방적이면서도 근거 있는 환경으로 보고, 과제 무결성 검증과 안정적 학습을 위한 검증 가능 피드백 모두를 제공하게 한다. AZR이 프로그램·입력·출력 삼중쌍의 특정 요소에 대한 추론을 요구하는 과제를 구성하게 하며, 이는 귀납(induction)·귀추(abduction)·연역(deduction)이라는 세 가지 추론 모드에 대응한다. 제안 접근의 다중 과제 특성에 맞춘 새로운 RL 어드밴티지 추정기로 전체 시스템을 end-to-end 학습한다. 도메인 내 데이터 없이도 AZR은 수학·코딩에서 뛰어난 능력을 보이며, 코딩에서는 새로운 SOTA를 세운다. (이하 발견 사항: 코드 사전지식이 추론을 증폭, 교차 도메인 전이가 더 뚜렷, 큰 베이스가 더 큰 이득, ReAct 유사 행동 자연 발현, 추론 모드별 인지 행동·토큰 길이 차이, 안전 경보 등 — 압축)

## 한 줄 핵심
외부 데이터 없이 모델이 코드 실행기를 환경 삼아 스스로 추론 과제를 제안하고 풀며 검증 가능 보상으로 자기 진화하는, "데이터 0" 자기 대국 RLVR 패러다임.
