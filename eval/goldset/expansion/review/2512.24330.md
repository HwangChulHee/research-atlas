# 2512.24330 SenseNova-MARS: Empowering Multimodal Agentic Reasoning and Search via Reinforcement Learning

> section_tag: RAG강화추론  ·  survey_source: s3

## 초록 (한글)
시각-언어 모델(VLM)은 에이전트적 추론으로 복잡한 과제를 풀 수 있지만, 그 능력은 대체로 텍스트 중심의 사고연쇄(chain-of-thought)나 고립된 도구 호출에 머물러 있다. 이들은 검색과 이미지 자르기(crop) 같은 외부 도구를 조율해야 하는 지식 집약적·시각적으로 복잡한 상황에서, 동적 도구 조작과 연속적 추론을 매끄럽게 교차하는 인간 수준의 숙련도를 보이지 못한다. 본 연구는 강화학습(RL)을 통해 VLM에 시각 추론과 도구 사용을 교차하는 능력을 부여하는 새로운 멀티모달 에이전트 추론·검색 프레임워크 SenseNova-MARS를 제안한다. 구체적으로 SenseNova-MARS는 이미지 검색, 텍스트 검색, 이미지 자르기 도구를 동적으로 통합해 세밀하고 지식 집약적인 시각 이해 문제를 다룬다. RL 단계에서는 학습 안정성을 높이고 도구 호출·추론 능력을 향상시키는 Batch-Normalized Group Sequence Policy Optimization(BN-GSPO) 알고리즘을 제안한다. 또한 복잡한 시각 과제에서 에이전트 VLM을 종합 평가하기 위해, 고해상도 이미지와 지식 집약·검색 기반 질문으로 구성된 최초의 검색 지향 벤치마크 HR-MMSearch를 도입한다. 실험 결과 SenseNova-MARS는 오픈소스 검색 및 세밀한 이미지 이해 벤치마크에서 SOTA를 달성한다. SenseNova-MARS-32B는 MMSearch에서 74.3, HR-MMSearch에서 54.4를 기록해 Gemini-3-Pro, GPT-5.2 같은 상용 모델을 능가한다. 모든 코드·모델·데이터셋을 공개할 예정이다.

## Introduction (한글)
VLM은 다양한 시각 이해 과제에서 인상적 능력으로 AGI 발전을 이끌었다. 최근에는 지식 집약 과제용 검색 엔진이나 세밀 분석용 자르기 도구 등 외부 도구와 추론 능력을 VLM에 결합하려는 노력이 있었으나, 이런 시스템들은 여전히 텍스트 중심 추론 사슬과 고립된 도구 호출에 갇혀 멀티모달 추론과 도구 사용의 동적 통합이 부족하다. 실제 시각 이해는 계획·추론·다중 도구 실행을 하나의 응집적·적응적 다단계 과정으로 교차할 수 있는 에이전트 모델을 요구한다.

DeepSeek-R1처럼 GRPO를 쓴 종단간(end-to-end) RL이 추론 능력 강화의 주류로 자리잡았다. 기존 검색 기반 에이전트들은 텍스트 검색 도구로 다회전 추론을 종단간 RL로 강화해 RAG 대비 큰 향상을 보였다. MMSearch-R1은 이미지 검색과 텍스트 검색을 통합해 VLM이 실제 환경과 상호작용하며 지식 집약·정보 탐색 시각 과제를 다루게 했다. 그러나 고해상도 인지 과제에는 여전히 부족한데, 이는 이미지·텍스트 검색만으로는 불가능한 세밀한 시각 추론을 요구한다. OpenAI-o3의 "이미지로 사고하기(Thinking with images)" 패러다임은 자르기 도구로 이미지·텍스트 추론을 교차하며, 후속 연구인 Pixel Reasoner와 DeepEyes는 픽셀 공간 시각 추론을 RL로 유도할 수 있음을 보였다. 그러나 이 접근들은 보통 검색 기반 또는 이미지 조작 도구 중 하나에 한정되어, 둘 다 필요한 동적 실세계 과제를 다루지 못한다.

이를 위해 저자들은 이미지 검색·텍스트 검색·이미지 자르기 도구를 동적 다회전 추론 과정에 통합하는 SenseNova-MARS를 제안한다. Qwen2.5-VL과 Qwen3-VL 위에 7B/8B/32B 세 규모로 구축하며, (1) 학습 데이터 구성, (2) 콜드스타트(SFT)+RL 2단계 학습, (3) RL 알고리즘 설계의 세 측면에 집중한다. 약 3,000개 샘플의 콜드스타트로 기본 도구 사용 패턴을 익히고, RL 단계에서 표준 GSPO를 확장한 BN-GSPO로 다중 도구 협업·추론을 강화한다. 또한 고해상도·검색 지향·지식 집약 과제 평가용 HR-MMSearch를 도입한다. ## 한 줄 핵심
시각-언어 모델이 텍스트/이미지 검색과 이미지 자르기 도구를 추론 도중 자유롭게 번갈아 쓰도록 RL로 훈련해, 고해상도·지식 집약 시각 질문에서 상용 모델을 능가하는 멀티모달 검색 에이전트를 만든다.
