# 2505.14246 Visual Agentic Reinforcement Fine-Tuning

> section_tag: RAG강화추론  ·  survey_source: s3

## 초록 (한글)
대규모 추론 모델(예: OpenAI의 o3)의 핵심 추세는 웹 브라우저로 검색하거나 코드를 작성·실행해 이미지를 다루며 "이미지로 사고(think with images)"하는 등 외부 도구를 사용하는 네이티브 에이전트 능력이다. 오픈소스 연구 커뮤니티에서는 함수 호출, 도구 통합 같은 언어 전용 에이전트 능력에서 큰 진전이 있었지만, 진정으로 이미지로 사고하는 멀티모달 에이전트 능력과 그 벤치마크 개발은 아직 덜 탐구되었다. 본 연구는 대규모 비전-언어 모델(LVLM)에 유연하고 적응적인 추론 능력을 부여하는 Visual Agentic Reinforcement Fine-Tuning(Visual-ARFT)의 효과를 부각한다. Visual-ARFT로 오픈소스 LVLM은 실시간 정보 갱신을 위해 웹사이트를 탐색하고, 입력 이미지를 자르기·회전 등으로 조작·분석하는 코드를 작성하는 능력을 얻는다. 또한 LVLM의 에이전트 검색·코딩 능력을 평가하기 위한 두 설정(MAT-Search, MAT-Coding)의 Multi-modal Agentic Tool Bench(MAT)를 제시한다. 실험 결과 Visual-ARFT는 MAT-Coding에서 +18.6% F1 / +13.0% EM, MAT-Search에서 +10.3% F1 / +8.7% EM로 기준선을 능가하고 최종적으로 GPT-4o를 넘어선다. 2Wiki, HotpotQA 같은 기존 다중 홉 QA에서도 +29.3% F1 / +25.9% EM 향상을 보여 강한 일반화를 입증한다.

## Introduction (한글)
최근 대규모 추론 모델(LRM)의 발전으로, 추론·계획하고 외부 도구와 상호작용해 복잡 과제를 푸는 새로운 세대의 에이전트 시스템이 등장했다. 그중 OpenAI의 o3는 텍스트·비주얼 양 모달리티에 걸친 도구 증강 추론을 네이티브로 지원해 큰 도약을 보였다. 이런 능력은 정적·단일 턴 추론에서 동적·다단계 의사결정으로의 전환을 의미하며, 웹 탐색·코드 실행·이미지 조작으로 실제 과제를 수행한다.

지도 미세조정(SFT)이 도구 사용 활성화의 주류 접근이었지만, 큐레이션된 시연이나 수작업 궤적에 의존해 확장·일반화가 어렵다. 그러나 강화 미세조정(RFT)으로 에이전트 시스템을 학습하는 추세가 커지고 있다. OpenAI-o1은 소수 예시만으로 추론 모델을 새 도메인에 빠르게 적응시키는 핵심 기법으로 RFT를 부각했다. o1의 세부는 비공개이나, DeepSeek-R1은 학습된 보상 모델이 아닌 규칙 기반 정확성 검사에서 도출된 검증 가능 보상 신호가 RFT의 효과적 감독 신호가 될 수 있음을 보였다.

Search-R1, ReTool 같은 최근 연구는 RFT를 에이전트 학습에 성공적으로 적용해, 모델이 검증 가능 보상으로 도구 사용 능력을 습득하게 했고 QA·수학 추론에서 유망한 결과를 보였다. 그러나 현 노력은 대체로 언어 전용 설정에 국한되어, 시각 입력과 도구 실행을 포함한 멀티모달 에이전트 추론에서 RFT의 잠재력은 거의 탐구되지 않았다.

이 격차를 해결하기 위해, LVLM에 에이전트 추론·도구 사용 능력을 부여하는 효과적 프레임워크 Visual-ARFT를 제안한다. 수작업 프롬프트나 대규모 지도 데이터에 의존하는 기존 접근과 달리, 보상 주도 학습 전략으로 LVLM이 추론하고 과제를 분해하며 필요 시 도구를 호출하도록 가르친다. 에이전트 추론·도구 사용 행동에 특화된 검증 가능 보상 집합을 설계하고, Group Relative Policy Optimization(GRPO) 알고리즘으로 보상 피드백에 기반해 정책을 갱신한다. 에이전트 검색과 에이전트 코딩이라는 두 도전적 멀티모달 설정에 집중한다. 또 학습·평가를 위해 MAT(MAT-Search, MAT-Coding)를 도입한다. (이하 기여 압축)

## 한 줄 핵심
규칙 기반 검증 가능 보상과 GRPO로 비전-언어 모델을 강화 미세조정해, 웹 검색과 이미지 조작 코드 작성·실행으로 "이미지로 사고"하는 멀티모달 에이전트 능력을 부여하는 방법(Visual-ARFT).
