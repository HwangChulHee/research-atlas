# 2505.04253 LLM-Independent Adaptive RAG: Let the Question Speak for Itself

> section_tag: 질의계획  ·  survey_source: s1

## 초록 (한글)
LLM은 환각에 취약하며, 검색 증강 생성(RAG)이 이를 완화하지만 높은 계산 비용이 들고 잘못된 정보를 들여올 위험이 있다. 적응형 검색(adaptive retrieval)은 필요할 때만 검색하는 것을 목표로 하지만, 기존 접근들은 LLM 기반 불확실성 추정에 의존해 비효율적이고 비실용적이다. 본 연구에서는 외부 정보(external information)에 기반한 경량의 LLM 비의존(LLM-independent) 적응형 검색 방법을 도입한다. 7개 그룹으로 조직된 27개 특징(feature)과 그 혼합 조합을 조사했다. 6개 QA 데이터셋에서 이 방법들을 평가하여 QA 성능과 효율성을 측정했다. 결과는 본 접근이 복잡한 LLM 기반 방법의 성능에 필적하면서도 상당한 효율 향상을 달성함을 보여, 적응형 검색에 외부 정보를 활용할 잠재력을 입증한다.

## Introduction (한글)
LLM은 QA 같은 과제에서 뛰어나지만 환각에 취약하다. RAG는 외부 정보를 통합해 이를 완화하나, 오류 누적과 외부 환각 같은 위험을 도입한다. 적응형 검색 기법들은 불확실성을 추정해 검색이 필요한지 판단함으로써 LLM 내부 지식과 외부 자원의 균형을 맞추려 한다. 그러나 기존 방법들은 주로 이 과제를 LLM 내부 상태나 출력 기반의 불확실성 추정으로 틀짓기 때문에 상당한 계산 오버헤드가 발생한다. 이는 검색 호출 감소로 얻는 효율 이득을 상쇄하고, 특히 더 큰 모델에서 실용성을 제한한다.

본 연구는 엔티티 인기도(entity popularity), 질문 유형(question type) 같은 외부 정보를 활용하는 LLM 비의존 적응형 검색 방법을 도입해 이 문제를 다룬다. 본 방법들은 비슷한 품질을 달성하면서도 훨씬 효율적이고 LLM을 전혀 필요로 하지 않는다. 평가 결과(Figure 1), 제안 특징들은 PFLOPs와 LLM 호출 측면에서 훨씬 효율적이며 하위 성능은 다른 적응형 검색 방법과 비슷하다. 기여는 (1) LLM 비의존 적응형 검색을 위한 27개 특징으로 구성된 7개 그룹의 경량 외부 정보 특징 도입, (2) LLM 기반 불확실성 추정을 없애 효율을 크게 개선하면서 QA 성능 유지, (3) 복잡한 질문에서 불확실성 기반 적응형 검색 방법을 능가함을 입증한다.

## 한 줄 핵심
LLM 불확실성 추정 대신 엔티티 인기도·질문 유형 같은 가벼운 외부 특징으로 검색 필요 여부를 결정해, 동등한 QA 성능을 훨씬 적은 계산으로 달성하는 LLM 비의존 적응형 RAG.
