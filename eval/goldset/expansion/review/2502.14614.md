## 초록 (한글)
외부 지식을 통합하는 검색증강 LLM은 임상 진단을 포함한 의료 도메인에서 뛰어난 성능을 보였다. 그러나 기존 RAG 방법은 진단 난이도와 입력 샘플의 정보성에 맞춰 검색 전략을 조정하는 데 어려움을 겪는 경우가 많다. 이 한계는 과도하고 불필요한 검색을 초래하여 계산 효율을 해치고, 진단 정확도를 떨어뜨릴 수 있는 잡음 도입 위험을 높인다. 이를 해결하기 위해 우리는 ICA-RAG(정보 완전성 안내 적응형 검색증강생성)를 제안하며, 이는 질병 진단에서 RAG 신뢰성을 높이는 새로운 프레임워크다. ICA-RAG는 적응형 제어 모듈을 사용해 입력의 정보 완전성에 기반하여 검색의 필요성을 평가한다. 검색을 최적화하고 지식 필터링을 통합함으로써, ICA-RAG는 검색 작업을 임상 요구에 더 잘 정렬한다. 세 개의 중국어 전자의무기록 데이터셋 실험은 ICA-RAG가 베이스라인 방법을 크게 능가하여 임상 진단에서의 효과를 입증함을 보여준다.

## Introduction (한글)
LLM은 임상 진단을 포함한 의료 과제에서 뛰어난 능력을 보였으나, 환각(그럴듯하지만 틀린 정보 생성)과 지식 갱신의 자원 집약성 같은 도전에 직면한다. RAG는 신뢰할 수 있는 외부 문서를 통합해 환각을 줄이고 최신 정보를 보장하는 해법을 제공한다.

연구자들이 고위험 도메인에서 LLM 정확도를 높이기 위해 RAG를 널리 탐구했지만, 모든 의료 사례가 이 접근을 필요로 하는 것은 아니다. 증상이 경미하고 진단이 명확한 흔한 질병은 검색 없이도 정확히 다룰 수 있다. 그러나 대부분의 기존 RAG 방법은 선택적 검색 논리가 없어 모든 질의에 무차별적으로 검색을 수행한다. 이는 계산·시간 비용을 늘릴 뿐 아니라 저품질 검색으로 오류를 도입해 성능을 오히려 저하시킬 수 있다.

검색 시스템의 효율을 높이기 위해 연구자들은 적응형 RAG 패러다임을 제안했으며, 특정 조건이 충족될 때만 검색을 활성화하는 제어 논리를 둔다. 두 가지 흔한 접근이 있다: (1) LLM의 출력 또는 확률 분포에 기반해 판단 조건 설정, (2) 더 작은 판단 모델을 훈련해 낮은 비용으로 검색 여부 결정. 그러나 전자는 LLM이 관련 지식이 부족해도 고확신 분포를 내는 과확신 경향과 로짓 접근 필요(API/폐쇄형 모델에 부적합) 한계가 있다. 후자는 입력 콘텐츠 특성에 크게 의존한다. 예컨대 Jeong et al.은 단일홉 질의를 "쉬운 질문", 멀티홉 질의를 "어려운 질문"으로 정의하는데, 이러한 QA 과제는 난이도 구배가 뚜렷해 모델이 상대적으로 쉽게 구별한다.

단일홉·멀티홉 QA와 달리, 의료 도메인의 입력 텍스트는 포착 가능한 뚜렷한 구조적 패턴이 없어 작은 언어모델이 답변 난이도를 이해하기 매우 어렵다. 따라서 이 접근의 성공 경험은 다른 과제로 직접 이전될 수 없다.

위 분석에 기반하여 우리는 질병 진단 접근 ICA-RAG를 제안한다. 복잡한 구조의 긴 의료 텍스트에 특화된 적응형 검색 결정 최적화를 사용한다. 핵심 혁신은 입력 정보 완전성에 기반한 검색 결정 최적화 모듈이다. 이 모듈은 긴 입력을 텍스트 단위로 분할하고, 분류 모델로 각 단위의 중요도를 예측하며, 전역 정보 완전성을 계산해 검색 필요성을 결정한다. 분류기가 이미 중요 텍스트 단위를 식별하므로 검색 시 이를 우선시하여 무관 정보의 간섭을 최소화한다. 단일 예측 라운드로 검색 결정 최적화와 질의 선택을 함께 달성한다.

## 한 줄 핵심
의료 입력 텍스트를 단위로 쪼개 분류기로 중요도·정보 완전성을 측정하고, 이를 기준으로 검색 여부를 적응적으로 결정·정련하는 질병 진단용 적응형 RAG 방법이다.
