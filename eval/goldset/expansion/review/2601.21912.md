# 2601.21912 ProRAG: Process-Supervised Reinforcement Learning for Retrieval-Augmented Generation

> section_tag: 학습형RL  ·  survey_source: s3

## 초록 (한글)
강화학습(RL)은 복잡한 추론 과제에서 검색증강생성(RAG)을 최적화하는 유망한 패러다임이 되었다. 그러나 전통적인 결과 기반(outcome-based) RL 방식은 종종 보상 희소성과 비효율적 신용 할당(credit assignment) 문제를 겪는다. 거친 스칼라 보상으로는 긴 호흡(long-horizon) 궤적 안에서 어떤 단계가 잘못되었는지 짚어낼 수 없기 때문이다. 이 모호함은 흔히 "프로세스 환각(process hallucination)"을 낳는데, 모델이 잘못된 논리나 불필요한 검색 단계를 거치고도 정답에 도달하는 현상이다. 최근의 프로세스 인지 방식들은 정적 선호 학습이나 휴리스틱 보상 정형화로 이를 완화하려 하지만, 단계별 신용을 전역 결과로부터 분리하는 데 필요한 온폴리시(on-policy) 탐색 능력이 부족한 경우가 많다. 이를 해결하기 위해 우리는 학습된 단계별 감독을 온라인 최적화 루프에 통합하는 프로세스 감독 강화학습 프레임워크 ProRAG를 제안한다. 프레임워크는 네 단계로 구성된다: (1) 구조화된 추론 형식으로 모델을 초기화하는 지도 정책 웜업, (2) 중간 추론 품질을 정량화하는 MCTS 기반 프로세스 보상 모델(PRM) 구축, (3) 정책을 세밀한 프로세스 선호에 정렬하는 PRM 유도 추론 정제, (4) 이중 입도(dual-granularity) 어드밴티지 메커니즘을 갖춘 프로세스 감독 강화학습. 단계별 프로세스 보상을 전역 결과 신호와 결합함으로써 ProRAG는 모든 행동에 정밀한 피드백을 제공한다. 다섯 개 멀티홉 추론 벤치마크 실험에서 ProRAG는 강력한 결과 기반·프로세스 인지 RL 기준선보다, 특히 복잡한 긴 호흡 과제에서 우월한 성능을 달성하여 세밀한 프로세스 감독의 효과를 입증했다. 코드와 모델은 공개되어 있다.

## Introduction (한글)
최근 LLM은 뛰어난 추론 능력을 보이며 복잡하고 지식 집약적인 과제에서 성능을 크게 끌어올렸다. 이러한 발전은 RAG를 정적 파이프라인에서 에이전트형 시스템으로 진화시켰는데, 여기서 모델은 검색 전략을 능동적으로 계획하고 외부 도구와 상호작용하며 피드백에 따라 추론을 반복적으로 정제하는 자율적 의사결정자로 행동한다. 그러나 이러한 동적 워크플로우를 표준 지도학습으로 최적화하기는 어렵다. 효과적인 추론·검색 궤적은 열린 형태인 경우가 많아 중간 결정에 대한 세밀한 정답 감독을 얻기 어렵거나 비용이 과도하다. 이 간극을 메우기 위해 최종 결과 신호로 정책을 최적화하는 RL이 점점 채택되고 있다.

그러나 전통적 결과 기반 RL은 보상 희소성과 신용 할당 문제로 복잡한 멀티홉 RAG에 비효과적이다. 긴 호흡 궤적에서 최종 정답 정확도에 기반한 거친 스칼라 보상은 중간 행동의 구체적 기여를 구별하지 못한다. 이로 인해 "프로세스 환각"이 생기며, 결과 지향 피드백은 오히려 진정한 추론 능력 대신 가짜 상관관계를 강화할 수 있다.

이 문제에 대해 최근 연구는 프로세스 수준 신호를 도입하는 최적화 전략을 탐구하며 오프라인·온라인 방식으로 나뉜다. 오프라인 방식은 MCTS 같은 탐색 알고리즘으로 선호 궤적을 합성해 DPO에 활용하지만, 정적 데이터 의존으로 온폴리시 분포 변화에 적응하기 어렵다. 온라인 방식은 휴리스틱 보상 정형화, 단계별 정규화, 멀티에이전트 상호작용에 기대지만 주로 보상 크기 조절이나 비싼 LLM 심판에 의존할 뿐 중간 단계의 논리적 타당성을 검증하지는 못한다. 따라서 온라인 학습 중 직접 밀집(dense)·탐색 유도 프로세스 감독을 제공하는 것은 여전히 어렵다.

본 논문에서 우리는 멀티홉 RAG의 신용 할당 문제를 해결하기 위해 프로세스 감독 강화학습 프레임워크 ProRAG를 제안한다. 기존 방식과 달리 ProRAG는 학습된 단계별 감독을 온라인 최적화 루프에 직접 통합한다. 구체적으로 지도 정책 웜업으로 시작해, 이 정책을 사전 정보로 삼아 MCTS로 구성한 다양한 추론 경로에 PRM을 학습시킨다. 초기 정책과 평가기 간 정렬 간극을 메우기 위해 추론 정제 단계를 도입하고, 마지막으로 PRM의 단계별 프로세스 어드밴티지와 전역 결과 보상을 결합하는 이중 입도 메커니즘의 프로세스 감독 RL 단계를 구현한다. (압축함)

PopQA, HotpotQA, 2WikiMultihopQA, MuSiQue, Bamboogle 다섯 벤치마크 실험에서 ProRAG는 강한 기준선을 능가했고, 절제 연구가 각 구성요소의 기여를 확인했다. 또한 ProRAG는 검색 과정을 효과적으로 조절하고, 데이터 효율성과 낮은 추론 지연을 달성했다.

## 한 줄 핵심
MCTS 기반 프로세스 보상 모델로 만든 단계별 보상을 전역 결과 보상과 이중 입도로 결합해, 멀티홉 RAG의 신용 할당과 "프로세스 환각" 문제를 온라인 RL로 푸는 프레임워크.
