# 2604.17555 CoSearch: Joint Training of Reasoning and Document Ranking via Reinforcement Learning for Agentic Search

> section_tag: 질의계획;학습형RL  ·  survey_source: s3

## 초록 (한글)
에이전트형 검색—반복적으로 추론하고 질의를 발행하며 검색된 정보를 종합해 복잡한 질문에 답하도록 에이전트를 훈련하는 과제—은 강화학습(RL)을 통해 큰 진전을 이뤘다. 그러나 Search-R1 같은 기존 접근은 검색 시스템을 고정된 도구로 취급해 추론 에이전트만 최적화하고 검색 구성요소는 그대로 둔다. 예비 실험에서 오라클 검색과 고정 검색 간 격차가 일곱 개 QA 벤치마크에 걸쳐 상대 F1 +26.8%에 달해, 검색 시스템이 에이전트형 검색 성능 확장의 핵심 병목임을 시사한다. 이에 착안해, 멀티스텝 추론 에이전트와 생성형 문서 랭킹 모델을 GRPO로 공동 훈련하는 프레임워크 COSEARCH를 제안한다. 추론 궤적마다 입력이 달라지는 랭커의 효과적 GRPO 훈련을 위해, 서브질의를 토큰 수준 유사도로 군집화해 추가 롤아웃 없이 유효 최적화 그룹을 형성하는 의미 그룹화 전략을 도입한다. 또 랭킹 품질 신호와 궤적 수준 결과 피드백을 결합한 복합 보상을 설계해 랭커에 즉각·장기 학습 신호를 모두 제공한다. 일곱 개 단일홉·멀티홉 QA 벤치마크 실험에서 강한 베이스라인을 일관되게 능가하며, 절제 연구가 각 설계 선택을 검증한다. 추론 에이전트와 검색 시스템의 공동 훈련이 가능하고 강력함을 보여 미래 검색 에이전트의 핵심 요소를 제시한다.

## Introduction (한글)
LLM은 강한 추론 능력을 보이지만 파라미터 학습 데이터를 넘어서는 지식이 필요할 때 근본적 한계에 직면한다. RAG는 검색을 통해 외부 지식 접근을 허용해 이를 해결한다. 초기에는 단일 턴 검색 후 답변 생성에 의존했으나, 최근 연구는 모델이 반복적으로 추론·검색 질의 생성·검색 정보 통합을 수행하는 멀티스텝 에이전트형 검색이 지식 집약적 과제에서 훨씬 강함을 보였다. 이런 시스템에서 에이전트는 필요 정보를 추론하고 질의를 발행하며 검색 문서를 관찰하는 과정을 충분한 증거가 모일 때까지 반복한다. 최종 정답성 기반 보상으로 RL 훈련하는 것이 언제·무엇을 검색할지 가르치는 데 효과적임이 입증됐다(Jin et al. 2025 등). 그러나 이들 시스템의 공통 가정은 검색 모델/검색 엔진이 고정된다는 것으로, 에이전트는 검색 모델을 도구로 취급해 사용법만 배우고 도구는 적응하지 않는다.

이 가정이 큰 병목을 만든다고 주장한다. RL 훈련 검색 에이전트 분석에서, 대부분의 오류는 추론이 아니라 검색 구성요소에서 비롯된다—에이전트는 그럴듯한 질의를 발행하지만 검색 시스템이 관련 문서를 반환하지 못해, 질의를 재구성하거나 잡음에서 잘못된 결론을 내린다. 이를 정량화하고자 오라클 검색 실험(각 단계에서 정답 일치 문서를 상위로 승격)을 수행했고, 7B 에이전트는 평균 F1 +14.7%, 3B는 +26.8% 개선되어 추론 능력과 무관하게 검색 병목이 광범위함을 보였다. 이는 "멀티턴 추론 에이전트와 함께 검색 시스템을 공동 최적화해 서로 보완하도록 학습시킬 수 있는가?"라는 질문으로 이어진다.

본 연구는 주 추론 에이전트와 생성형 랭커를 공동 훈련하는 RL 프레임워크 COSEARCH를 제안한다. 주 에이전트는 표준 ReAct 스타일 루프로 사고와 검색 질의를 생성한다. 생성형 랭커는 주 에이전트의 서브질의와 고정 1단계 밀집 검색기의 후보 문서를 받아 부분집합을 선택·재랭킹한다. 둘은 GRPO로 동시 훈련되며 보상은 최종 정답성에서 도출된다. 랭커에 GRPO를 적용할 때 두 난제가 있다. 첫째, GRPO는 같은 프롬프트에서 나온 응답 그룹에 대해 어드밴티지를 계산하는데, 주 에이전트는 자연히 충족되나 랭커는 궤적마다 다른 서브질의를 받아 직접 그룹화가 불가하다—이를 토큰 수준 F1 유사도로 서브질의를 군집화하는 의미 그룹화로 해결한다. 둘째, 랭커의 최종 답 영향이 간접적이므로, 랭킹 품질 지표(Hit@k) 기반 관련성 보상과 궤적 수준 보상을 결합한 복합 보상을 설계한다. 두 보상 모두 규칙 기반 지표로 계산되어 비싼 LLM 주석이 불필요하다. Qwen2.5-7B-Instruct를 양쪽 백본으로 일곱 개 QA 벤치마크에서 평가해 Search-R1과 고정 랭커 변형을 일관되게 능가한다(7B에서 Search-R1 대비 상대 F1 +6.6%, 3B에서 +10.8%).

## 한 줄 핵심
검색기를 고정 도구로 두는 대신 추론 에이전트와 생성형 문서 랭커를 GRPO로 함께 훈련해 검색 병목을 직접 줄이는 에이전트형 검색 프레임워크.
