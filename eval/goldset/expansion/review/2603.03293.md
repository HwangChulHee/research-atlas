# 2603.03293 SE-Search: Self-Evolving Search Agent via Memory and Dense Reward

> section_tag: 학습형RL  ·  survey_source: s3

## 초록 (한글)
검색증강생성(RAG)은 검색된 외부 지식에 생성을 조건화하여 LLM의 환각과 사실 오류를 줄인다. 최근 검색 에이전트는 RAG를 자율적 다중 턴 정보 탐색 과정으로 더 확장한다. 그러나 기존 방법은 종종 무관하거나 잡음 섞인 문서를 축적하고, 희소한 강화학습 신호에 의존한다. 우리는 메모리 정화(memory purification), 원자 질의 훈련(atomic query training), 밀집 보상(dense rewards)의 세 구성요소로 온라인 검색 행동을 개선하는 자기진화(Self-Evolving) 검색 에이전트 SE-Search를 제안한다. SE-Search는 두드러진 증거를 유지하면서 무관한 내용을 걸러내는 Think-Search-Memorize 전략을 따른다. 원자 질의 훈련은 더 짧고 다양한 질의를 촉진해 증거 획득을 개선한다. 밀집 보상은 세밀한 피드백을 제공해 훈련을 가속한다. 단일홉·멀티홉 QA 벤치마크 실험에서 SE-Search-3B는 강한 기준선을 능가하여 Search-R1 대비 절대 10.8점, 상대 33.8% 향상을 달성했다.

## Introduction (한글)
정보 검색은 일상의 흔한 활동이다. Google, Bing 같은 검색 엔진은 질의에 순위화된 웹 페이지를 반환해 관련 콘텐츠에 빠르게 접근하게 한다. 그러나 질의가 복잡해지고 사용자가 더 높은 정밀도를 기대하면서, 전통적 정보 검색 기법은 미묘한 의도를 포착하지 못하고 맥락에 맞지 않는 결과를 낼 수 있다.

LLM은 언어 이해, 추론, 정보 통합에서 강한 능력을 보였지만, 최신 외부 지식에 안정적으로 접근하지 못하고 사실 오류를 낼 수 있다. RAG는 검색된 외부 구절에 생성을 조건화해 이를 줄이지만, 고정 RAG 파이프라인은 LLM이 언제·무엇을 검색할지 결정하는 능력을 제한해 실세계 응용 상호작용 시 유연성이 떨어진다.

최근 연구는 사용자 의도를 해석하고 검색 전략을 계획하며 다중 턴 검색 행동을 수행하고 검색 증거를 축적하는 LLM 기반 에이전트를 제시한다. 이들은 검색 에이전트로 불린다. 대표적으로 Search-R1은 규칙 기반 보상의 강화학습으로 에이전트가 대규모 코퍼스에서 검색 도구와 상호작용하도록 훈련한다.

유망한 결과에도 기존 검색 에이전트 방법은 세 가지 핵심 난제에 직면한다. 잡음 섞인 검색 결과: 질의 후 top-K 문서를 검색하는데 다수가 무관·잡음이다. MAIN-RAG는 다중 LLM 필터링·점수화로 잡음을 줄이지만 훈련 없는(training-free) RAG 환경에서 작동한다. 제한된 검색 다양성과 미탐구된 검색 빈도: 선행 방법은 단계 간 유사 질의를 생성해 다양·유용한 증거 탐색을 제한한다. O2-Searcher는 임베딩 유사도 기반 질의 다양성 보상을 도입하나 추가 인코딩 단계가 효율을 떨어뜨린다. 희소한 진화적 피드백 신호: Search-R1은 최종 답 수준에서만 감독하고 질의 형성·형식·적절한 검색 빈도를 명시적으로 보상하지 않아, 에이전트가 질문 복잡도와 무관하게 지나치게 긴 질의를 생성하는 등 멀티홉에서 효과가 제한된다. TooRL은 더 풍부한 보상을 제안하나 검색 특화 행동이 아닌 일반 도구 사용을 겨냥한다.

다윈의 진화론(변화하는 환경에 대한 적응 강조)에서 영감을 받아, 우리는 보상·피드백 신호가 복잡한 실세계 질의에서 에이전트를 어떻게 안내하는지에 초점을 둔다. 이러한 신호는 사실을 직접 주지 않지만, 에이전트가 외부 정보를 발견·처리·사용하는 방식을 형성한다.

이에 우리는 LLM의 자율 검색 행동을 개선하는 자기진화 검색 에이전트 SE-Search를 제안한다. SE-Search는 검색 후 기억(memorize-after-search) 패러다임을 채택하고, 메모리 정화 템플릿으로 유용한 증거를 추출해 잡음 검색을 줄인다. 메모리 관리를 위해 보조 LLM에 의존하는 DeepAgent와 달리 SE-Search는 에이전트의 자기 메모리를 사용한다. 다양한 검색 행동과 적절한 검색 빈도를 촉진하기 위해, Atom-Searcher에서 동기를 얻은 원자 질의(Atomic Query) 전략을 도입해 여러 개의 구별되는 원자 질의를 생성하고 다단계 검색을 수행하게 한다. 마지막으로 Query, Memory, Outcome, Format 네 구성의 밀집 보상을 설계해 세밀한 RL 피드백을 제공하고 행동 규율을 개선하며 훈련을 안정화한다.

기여: SE-Search 제안, 메모리 정화·원자 질의·밀집 보상 세 메커니즘 도입, 일곱 개의 다양한 QA 벤치마크에서 효과·일반화 입증.

## 한 줄 핵심
검색 에이전트의 잡음 문서·희소 보상 문제를, Think-Search-Memorize 메모리 정화 + 짧고 다양한 원자 질의 + 4종 밀집 보상으로 푸는 자기진화형 검색 에이전트(GRPO 기반).
