# 2504.16787 Credible plan-driven rag method for multi-hop question answering

> section_tag: 질의계획  ·  survey_source: s1

## 초록 (한글)
검색 증강 생성(RAG)은 외부 지식을 LLM에 통합함으로써 단일 홉 질의응답에서 강한 성능을 보여왔다. 그러나 안정적 추론과 사실 일관성을 모두 요구하는 다중 홉 QA에서는 효과가 제한적이다. 기존 접근들은 추론 궤적 안정성이나 사실 검증 중 하나만 다루는 부분적 해법을 제공할 뿐, 둘을 동시에 달성하는 경우는 드물다. 이 격차를 메우기 위해, PDCA 사이클에서 영감을 받은 3단계 Plan-then-Act-and-Review 프레임워크인 PAR-RAG를 제안한다. PAR-RAG는 의미적 복잡도(semantic complexity)를 통합 원리로 삼아 세 핵심 구성요소를 갖는다. (i) 복잡도 인식 예시 선택은 분해 세분성을 질문 난이도에 맞춰 계획 생성을 유도해 추론 궤적을 안정화한다. (ii) 실행은 구조화된 retrieve-then-read 과정을 따른다. (iii) 이중 검증(dual verification)은 중간 오류를 식별·교정하면서, 질문 복잡도에 따라 검증 강도를 동적으로 조정한다(단순 질의는 정확도 강조, 복잡 질의는 다중 증거 일관성 강조). 다양한 벤치마크 실험에서 PAR-RAG는 경쟁 기준선을 일관되게 능가했고, 절제 연구는 복잡도 인식 계획과 이중 검증의 상호 보완적 역할을 확인했다.

## Introduction (한글)
RAG는 외부 지식으로 LLM을 강화하는 유망한 패러다임이지만, 여러 출처에서 증거를 검색해 다단계 추론으로 통합해야 하는 다중 홉 QA(MHQA)에서는 여전히 미흡하다. 두 가지 결합된 난제가 있다. (1) 추론 경로 이탈(reasoning path deviation): 중간 단계가 의도된 추론 궤적에서 벗어남, (2) 중간 사실 오류(intermediate factual errors): 환각이나 무관한 맥락에서 비롯됨. 이 오류들은 단계 간 전파되어 불일치를 누적시키고 최종 답의 신뢰성을 떨어뜨린다.

선행 연구는 여러 보완적 방향을 탐색했다. 계획 유도 방법은 명시적 과제 분해로 추론을 안정화하고, 자기 반성 접근은 검증 기제로 사실 정합을 높이며, 구조 인식 시스템은 그래프·계층 표현으로 문서 간 관계를 포착하고, 반복 추론 프레임워크는 반복 사이클로 중간 출력을 정제한다. 그러나 각 패러다임은 부분적 개선만 제공하고 통합 해법은 없다. 반성·추론 강화 접근은 사실 정확도를 높이지만 전역 추론 궤적을 불안정하게 하고, 계획 유도 방법은 궤적 안정성은 강화하나 검증이 약하거나 임시적이다. 구조 인식 시스템은 데이터 구조 최적화를 강조하나 견고한 추론 능력과 중간 결과 검증 기제가 모두 부족하다. 더 중요하게, 대부분의 방법이 의미적 복잡도의 역할을 간과한다. 계획 생성을 위한 예시 선택은 보통 정적이어서 추론 세분성과 문제 난이도가 어긋나고, 검증은 단일 정확도 점수로 뭉뚱그려진다.

이 격차를 메우기 위해, PDCA 사이클에서 영감받은 3단계 Plan-then-Act-and-Review 패러다임을 구현한 PAR-RAG를 제안한다. 의미적 복잡도를 추론 과정을 조직하는 통합 원리로 삽입한다. 계획에서는 추론된 복잡도에 매칭된 예시 사례가 계획 생성을 조건화해 분해 세분성을 과제 난이도에 맞춘다. 행동에서는 retrieve-then-read 단계로 계획을 실행한다. 검토에서는 이중 검증 모듈이 복잡도에 적응한다. 계획 유도 방법 계열에 속하지만, 의미적 복잡도가 주도하는 적응형 계획과 이중 검증을 도입함으로써 패러다임을 근본적으로 발전시킨다. (이하 비교·기여 압축)

## 한 줄 핵심
PDCA(계획-실행-검토) 사이클을 본떠, 질문의 의미적 복잡도에 따라 분해 세분성과 검증 강도를 조절하며 다중 홉 QA를 푸는 계획 주도 RAG 프레임워크.
