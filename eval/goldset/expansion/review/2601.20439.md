# 2601.20439 PEARL: Plan Exploration and Adaptive Reinforcement Learning for Multihop Tool Use

## 초록 (한글)
대형 언어 모델(LLM)은 외부 도구와 결합할 때 큰 잠재력을 보이지만, 복잡한 다회전 도구 호출에서 상당한 어려움을 겪는다. 약한 계획, 도구 환각(tool hallucination), 잘못된 파라미터 생성, 견고한 상호작용의 어려움 등을 흔히 보인다. 이를 해결하기 위해 정교한 도구 사용을 위한 LLM 계획·실행을 강화하는 새로운 프레임워크 PEARL을 제시한다. PEARL은 2단계 접근을 채택한다. 에이전트가 도구를 탐색해 유효 사용 패턴과 실패 조건을 학습하는 오프라인 단계, 그리고 온라인 강화학습 단계다. 온라인 단계에서는 계획 품질에 대한 뚜렷한 신호를 제공하는 정교히 설계된 보상 함수와 함께 Group Relative Policy Optimization(GRPO)으로 전담 Planner를 학습한다. ToolHop과 T-Eval 벤치마크 실험에서 PEARL은 기존 방법들을 크게 능가하여 ToolHop에서 56.5%의 새로운 SOTA 성공률을 달성하면서 낮은 호출 오류율을 유지한다. 본 연구는 도구 사용의 복잡한 계획 난제를 해결하는 핵심 진전으로, 더 견고하고 신뢰할 만한 LLM 기반 에이전트 개발에 기여한다.

## Introduction (한글)
LLM은 자연어 이해·생성에서 뛰어난 능력을 보이며 AI의 패러다임 전환을 이뤘다. 그러나 내재 지식이 정적이고 학습 데이터에 갇혀 있어 실세계 상호작용, 최신 정보 접근, 복잡 계산 능력이 제한된다. 이 한계의 변혁적 해법으로 LLM에 외부 도구를 결합하는 것(흔히 도구 학습, tool learning)이 등장했다. 이는 LLM이 API·데이터베이스·전문 함수에 접근하는 자율 에이전트로 기능하게 한다. Toolformer 같은 초기 성공은 모델이 단순한 단일 단계 API 호출을 학습할 수 있음을 보였다.

LLM 에이전트에 대한 야심이 커지면서 초점은 단순 도구 호출에서 여러 도구를 순차적으로 전략적으로 조율해야 하는 복잡한 다단계 문제 해결로 빠르게 이동하고 있다. 이 복잡도 도약은 결정적 병목, 즉 에이전트의 일관된 장기(long-horizon) 계획과 견고한 실행 능력을 드러낸다. 현재 접근들은 이 까다로운 영역에서 자주 실패한다. 많은 에이전트가 근시안적 한 번에 한 단계 전략을 채택해 장기 계획을 세우거나 지키지 못한다. 이런 통찰 부족은 존재하지 않는 도구 환각, 잘못된 파라미터 생성, 연속 도구 호출 간 의존성 관리 실패 등 취약하고 불안정한 실행을 낳는다. 게다가 적응 능력이 부족해, 불가피한 실행 오류에 직면해도 실패를 진단·반성하고 전략을 동적으로 조정하지 못해 반복적 실패 루프에 갇히곤 한다.

이를 위해 PEARL(Plan Exploration and Adaptive Reinforcement Learning for Tool Use)을 도입한다. 첫째, 견고한 실행을 위해 오프라인 도구 탐색 단계를 두어, 에이전트가 각 도구와 통제된 시행착오로 상호작용해 유효 사용 패턴·인자 형식·실패 조건을 학습한 "학습된 사용 설명서"를 만들어 실행 오류를 크게 줄인다. 둘째, 전략적 사고를 기르기 위해 전담 Planner를 적응적 RL로 학습하는 분리(decoupled) 구조를 채택한다. GRPO로 최적화된 Planner는 실행 전 완전한 고수준 계획을 생성하도록 학습하며, 핵심은 제안된 도구 사슬 품질에 직접적·밀집(dense) 피드백을 주는 계획 중심 보상 함수다. (intro 핵심 보존, 일부 압축)

## 한 줄 핵심
다단계 도구 사용에서 LLM 에이전트가 계획을 못 세우고 도구·파라미터를 환각하는 문제를, 오프라인 도구 탐색(사용 설명서 학습) + GRPO로 학습하는 전담 Planner(계획 중심 밀집 보상)의 2단계로 해결한다.
