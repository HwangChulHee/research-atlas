# 2505.15776 ConvSearch-R1: Enhancing Query Reformulation for Conversational Search with Reasoning via Reinforcement Learning

> section_tag: 질의계획;학습형RL  ·  survey_source: s2;s3

## 초록 (한글)
대화형 검색 시스템은 모호성·생략·대용어(coreference)를 흔히 포함하는 맥락 의존적 질의를 효과적으로 다뤄야 한다. 대화형 질의 재구성(CQR)은 이런 질의를 기성(off-the-shelf) 검색기에 적합한 독립적(self-contained) 형태로 변환해 이 문제를 해결한다. 그러나 기존 CQR 접근은 두 가지 결정적 제약이 있다. 인간 주석이나 LLM의 비싼 외부 감독에 대한 높은 의존, 그리고 재작성 모델과 하위 검색기 간 정렬 부족이다. 본 연구는 강화학습을 활용해 검색 신호로 직접 재구성을 최적화함으로써 외부 재작성 감독 의존을 완전히 제거하는 첫 자기 주도 프레임워크 ConvSearch-R1을 제시한다. 새로운 2단계 접근은 검색 유도 자기 증류로 cold-start 문제를 해결하는 Self-Driven Policy Warm-Up과, 기존 검색 지표의 희소성 문제를 다루는 특별 설계된 rank-incentive 보상 형성(reward shaping) 기제를 갖춘 Retrieval-Guided Reinforcement Learning을 결합한다. TopiOCQA와 QReCC 데이터셋 실험에서 ConvSearch-R1은 이전 SOTA를 크게 능가하며, 더 작은 3B 모델로 외부 감독 없이 도전적인 TopiOCQA에서 10% 이상 향상을 달성한다.

## Introduction (한글)
대화형 검색은 단일 턴 키워드 기반 전통 IR과 달리 다중 턴 상호작용으로 사용자 정보 요구를 충족한다. 이런 다중 턴 시나리오에서 대화형 질의는 모호성·생략·대용어를 자주 포함해 기존 검색 방법이 사용자 의도를 정확히 포착하기 어렵다. 다중 턴 검색기 학습의 어려움과 계산 비용을 고려해, 대화형 질의 재구성(CQR)이 맥락 의존 질의를 기성 검색기로 효과적으로 처리 가능한 독립적 형태로 변환하는 실용적 해법으로 떠올랐다.

기존 CQR 접근은 다양한 전략을 탐구했다. 일부는 인간 주석이나 ChatGPT 같은 LLM의 지식 증류로 얻은 명시적 재작성을 감독으로 사용한다. 더 최근 접근은 선호 최적화에 검색 신호를 활용하려 하지만, 여전히 초기화를 위해 외부 주석 데이터가 필요하고 일반적으로 실제 순위 개선을 최적화하기보다 쌍별(pairwise) 선호로 탐색을 제한한다. 이들은 두 가지 결정적 제약을 갖는다. (1) 고품질 주석을 위한 비싸고 일관성 없는 외부 출처에의 높은 의존, (2) 재작성 모델과 하위 검색기 간 정렬 부족. 근본 과제는 명시적 참조 재작성 없이, 검색 피드백이 안내하는 자기 탐색을 통해 질의 재구성 모델이 검색기와 효과적으로 정렬되게 하는 것이다.

이를 위해, 외부 재작성 감독 의존을 완전히 제거하는 자기 주도 프레임워크 ConvSearch-R1을 제안한다. 검색 순위 신호를 보상으로 활용해, 모델이 반복적 탐색·활용으로 효과적 재작성을 스스로 발견한다. 2단계 프레임워크를 설계한다. (1) Self-Driven Policy Warm-Up(SD-PWU): 모델의 few-shot 추론 능력과 검색 순위 신호를 결합해 외부 감독 없이 고품질 재작성 데이터를 자기 증류하여 cold-start 문제를 해결한다. (2) Retrieval-Guided Reinforcement Learning: Group Relative Policy Optimization(GRPO)으로 재작성 모델을 검색기와 더 정렬한다. 세심히 설계된 rank-incentive 보상 형성으로 Recall@K, NDCG@K 같은 기존 검색 지표의 희소성 문제를 다뤄, 이진적이거나 크게 치우친 결과 대신 더 매끄러운 학습 신호를 제공해 방대한 재구성 공간의 안정적·효율적 탐색을 가능케 한다. 3B 모델로 7B 모델 접근보다 더 좋은 성능을 내며, TopiOCQA·QReCC에서 SOTA를 달성한다. (이하 기여 압축)

## 한 줄 핵심
인간/LLM 재작성 감독 없이, 검색 순위 신호를 보상으로 삼아 RL(GRPO)로 대화형 질의 재구성 모델을 검색기와 정렬시키는 자기 주도 2단계 프레임워크.
