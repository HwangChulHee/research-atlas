# 2505.24332 Pangu DeepDiver: Adaptive Search Intensity Scaling via Open-Web Reinforcement Learning

> section_tag: 검색시점  ·  survey_source: s3

## 초록 (한글)
정보 탐색은 반복적인 증거 수집과 성찰적 추론을 요구하지만, 대형 언어 모델(LLM)은 오픈 웹 질의응답에서 여전히 이를 잘 수행하지 못한다. 기존의 프롬프팅 및 지도 미세조정(SFT) 방법은 프롬프트 규칙이나 학습 코퍼스에 고정되어 있으며, 보통 잘 정제된 위키 자료에서만 벤치마크되어 실제 적용성이 제한된다. 본 논문은 위키와 오픈 도메인 질의 모두에 걸쳐 실시간 인터넷에서의 정보 탐색을 평가하는 24k 학습 샘플·275 테스트 샘플 벤치마크인 WebPuzzle을 제시한다. 7k개의 WebPuzzle 인스턴스를 활용해, 과신·근거 부족 답변에 안주하는 대신 검색 빈도와 깊이를 끌어올리는 창발적 능력인 검색 강도 스케일링(Search Intensity Scaling, SIS)을 길러내는 강화학습(RL) 프레임워크 DeepDiver를 개발한다. SIS를 통해 Qwen2.5-7B-Instruct와 Pangu-7B-Reasoner는 실제 웹 과제에서 671B 규모의 DeepSeek-R1에 필적하는 성능을 달성한다. 콜드스타트 SFT에서 잘 설계된 RL 절차에 이르는 DeepDiver의 커리큘럼을 상세히 기술하며, 그 탐색 정책이 폐쇄형 질의에서 장문 작문과 같은 개방형 생성으로 일반화됨을 보인다. 본 연구 결과는 LLM의 적응형 정보 탐색을 진전시키고 향후 연구를 위한 엄밀한 벤치마크를 제공한다.

## Introduction (한글)
정보 탐색은 반복적 증거 수집, 성찰적 추론, 상충 정보 해소를 포함하는 근본적 인지 능력이다. AI의 큰 진전에도 LLM은 이러한 정보 탐색 행동을 재현하는 데 여전히 어려움을 겪으며, 언제·무엇을 탐색하고 증거의 관련성을 검증하며 잡음·상충 맥락에서 효과적으로 추론할지를 잘 결정하지 못한다.

반복적 RAG 프레임워크는 검색과 추론을 번갈아 수행해 이를 해결하려 한다. 기존 접근은 크게 두 부류다. (1) 프롬프팅 기반은 사전 정의된 규칙이나 ICL을 사용하지만, 프롬프트에 주어진 고정된 추론 흐름이 복잡·동적 문제에 대한 적응성을 제한한다. (2) 과제별 SFT는 검색·추론 능력을 향상시켜 더 나은 성능을 내지만, 학습 코퍼스에 묶인 추론 패턴을 내재화해 미지·동적 상황으로의 일반화가 제한된다.

최근 RL이 추론 시점 추론을 강화하는 데 적용되어 반복적 정교화와 탐색을 가능케 했다. 여러 연구가 RL을 반복적 RAG에 통합했으나, 이들은 대부분 위키피디아 기반의 HotpotQA처럼 잘 정제된 데이터셋에서 학습·평가된다. 그런 환경에서는 LLM 내부 지식만으로 많은 과제를 풀 수 있고 검색 환경이 "깨끗"해 잡음·상충이 거의 없다. 반면 실제 검색 환경은 잡음이 많고 일관성 없으며 신뢰할 수 없는 출처로 특징지어진다. 이 괴리는 보고된 "인센티브화된 검색 능력"의 일반화 가능성을 제한한다.

더 현실적인 오픈 도메인 시나리오에서 RL 유도 LLM 행동을 조사하기 위해 WebPuzzle을 도입한다(24k 학습, 275 인간 주석 테스트). 위키피디아로 풀 수 있는 과제와 오픈 웹에서 추출한 더 넓은 오픈 도메인 질의를 모두 포함하며, 위키 부분조차 외부 검색이 필요하도록 엄격히 검증된다. WebPuzzle과 함께, 이 데이터셋으로 학습한 RL 기반 탐색·추론 프레임워크 DeepDiver를 도입한다. DeepDiver는 실제 검색 엔진과 상호작용하며 검색 문서를 지속적으로 정교화·노이즈 제거한다. 핵심 혁신은 정보 수요 증가에 따라 검색 빈도와 깊이를 동적으로 확대하는 창발적 능력인 검색 강도 스케일링(SIS)이다.

체계적 실증 분석으로 검색 강도, 학습 환경, 일반화 능력 등 모델 행동에 영향을 주는 핵심 요인을 식별한다. 주요 통찰: (1) DeepDiver는 적응형 SIS로 뛰어난 정보 탐색 능력을 보이며 검색 깊이·빈도가 문제 난이도와 모델 성능에 비례한다. (2) "깨끗한" 위키 환경에 비해 WebPuzzle·실제 검색 환경이 복잡한 추론 행동(증거 보충, 상충 해소, 내용 검증, 자기교정 성찰)을 더 잘 유도한다. (3) RL 학습이 일반화 능력을 크게 향상시켜 폐쇄형에서 개방형 문제로의 전이를 가능케 한다.

## 한 줄 핵심
실제 오픈 웹 환경에서 검색 빈도·깊이를 문제 난이도에 맞춰 동적으로 키우는 능력(SIS)을 RL로 길러내고, 이를 평가할 WebPuzzle 벤치마크를 함께 제안한 논문.
