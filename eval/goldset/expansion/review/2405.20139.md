# 2405.20139 GNN-RAG: Graph Neural Retrieval for Large Language Model Reasoning

> section_tag: 질의계획;그래프RAG  ·  survey_source: s1

## 초록 (한글)
지식 그래프(KG)는 (머리, 관계, 꼬리) 삼중항 형태로 인간이 만든 사실 지식을 표현하며, 이들이 모여 그래프를 이룬다. KG 기반 질의응답(KGQA)은 자연어 질문에 대해 KG가 제공하는 정보에 추론을 근거 지어 답하는 과제다. LLM은 자연어 이해 능력 덕분에 QA의 최신(SOTA) 모델이지만, 그래프 신경망(GNN)은 KG에 저장된 복잡한 그래프 정보를 다룰 수 있어 KGQA에 널리 쓰여 왔다. 본 연구는 LLM의 언어 이해 능력과 GNN의 추론 능력을 RAG 방식으로 결합하는 새로운 방법 GNN-RAG를 제안한다. 먼저 GNN이 밀집 KG 서브그래프 위에서 추론해 질문에 대한 답 후보를 검색한다. 다음으로 질문 개체와 답 후보를 잇는 KG의 최단 경로를 추출해 KG 추론 경로로 표현한다. 추출된 경로를 언어화해 RAG와 함께 LLM 추론의 입력으로 준다. GNN-RAG에서 GNN은 밀집 서브그래프 추론기로서 유용한 그래프 정보를 추출하고, LLM은 자연어 처리 능력으로 최종 KGQA를 수행한다. 나아가 검색 증강(RA) 기법을 개발해 KGQA 성능을 더 높인다. 실험 결과 GNN-RAG는 WebQSP와 CWQ 두 벤치마크에서 SOTA를 달성했으며, 7B 튜닝 LLM으로 GPT-4 성능을 능가하거나 맞먹었다. 특히 다중홉·다중개체 질문에서 경쟁 기법을 답변 F1 기준 8.9~15.5%포인트 능가했다.

## Introduction (한글)
LLM은 자연어 이해 능력으로 많은 NLP 과제의 SOTA지만, 사전학습 비용 탓에 새 지식/도메인 지식에 적응하기 어렵고 환각에 취약하다. KG는 삼중항으로 사실 지식을 구조화 저장하며 갱신이 쉽고 다중홉 관계 등 복잡한 상호작용을 포착해 지식 집약 과제(QA)에 널리 쓰인다. RAG는 입력 문맥을 최신·정확 정보(예: KG에서 얻은)로 풍부하게 해 LLM 환각을 완화한다. KGQA에서 RAG 입력은 "지식: ... 질문: ..." 형태가 되어 LLM이 KG 정보에 접근해 답한다. RAG 성능은 검색된 KG 사실에 크게 좌우되는데, KG는 수백만 사실로 구성돼 효과적 그래프 처리가 필요하고 무관 정보 검색은 LLM을 혼란시킨다. LLM 기반 검색(LLM이 KG 정보 검색)은 복잡한 그래프 정보를 다루지 못하거나 GPT-4 같은 초대형 LM의 내부 지식에 의존해야 해 다중홉 KGQA에서 부진하다. GNN-RAG는 강력한 그래프 표현 학습기인 GNN을 검색에 재활용한다: (1) GNN이 밀집 KG 서브그래프에서 추론해 답 후보를 검색, (2) 질문 개체와 GNN 기반 답을 잇는 최단 경로를 추출해 KG 추론 경로로 언어화, RAG로 LLM에 입력. 또한 GNN-RAG를 LLM 기반 검색기로 증강(RA)해 성능을 더 높인다. 기여: (프레임워크) GNN을 KGQA 검색에 재활용해 LLM 추론 강화, (효과성·충실성) WebQSP/CWQ SOTA 및 복잡 질문 8.9~15.5% 향상, (효율) 추가 LLM 호출 없이 vanilla LLM 개선, 7B로 GPT-4 능가/맞먹음.

## 한 줄 핵심
GNN을 KG 서브그래프 추론·검색기로 재활용해 질문-답 최단 경로를 추출·언어화하고 이를 LLM에 RAG 입력으로 주는, 그래프 추론과 LLM을 결합한 KGQA 방법.

- RAG (Retrieval-Augmented Generation) — 근거: "Retrieval-augmented generation (RAG) is a framework that alleviates LLM hallucinations by enriching the input context ... e.g., obtained from the KG [Lewis et al., 2020]" — GNN-RAG는 RAG 방식으로 GNN 검색 결과를 LLM에 공급하는 패러다임 후속.
- GNN (Mavromatis and Karypis, 2022) — 근거: "GNN-RAG relies on Graph Neural Networks (GNNs) [Mavromatis and Karypis, 2022] ... GNN-RAG repurposes their graph processing power for retrieval" — 저자들의 선행 GNN 모델을 검색기로 적응·확장해 방법 자체를 세움(백본 적응형 계보 후보, 검증 필요).
