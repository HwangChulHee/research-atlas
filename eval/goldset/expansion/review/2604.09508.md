# 2604.09508 VISOR: Agentic Visual Retrieval-Augmented Generation via Iterative Search and Over-horizon Reasoning

> section_tag: 검색시점;학습형RL;RAG강화추론  ·  survey_source: s3

## 초록 (한글)
시각 검색증강생성(VRAG)은 시각 정보가 풍부한 문서를 검색·추론하도록 비전-언어 모델을 강화한다. 멀티스텝 추론이 필요한 복잡한 질의를 위해 에이전트형 VRAG는 추론과 반복 검색을 교차한다. 그러나 기존 에이전트형 VRAG는 두 가지 핵심 병목을 안고 있다. (1) 시각 증거 희소성: 핵심 증거가 여러 페이지에 흩어져 있는데 고립적으로 처리되어 페이지 간 추론을 방해하며, 이미지 내부의 미세 증거는 정밀한 시각 동작을 요구하는데 그 오용이 검색 품질을 떨어뜨린다. (2) 장기 지평에서의 검색 표류: 검색된 페이지의 시각 토큰이 누적되면서 맥락이 희석되고 인지 과부하를 유발해 에이전트가 검색 목표에서 이탈한다. 이를 해결하기 위해 단일 에이전트 프레임워크 VISOR를 제안한다. VISOR는 점진적 페이지 간 추론을 위한 구조화된 Evidence Space와 시각 동작을 관리하는 Visual Action Evaluation and Correction 메커니즘을 갖춘다. 또한 검색 표류 완화를 위해 슬라이딩 윈도우 기반 Dynamic Trajectory와 Intent Injection을 도입해, 이전 원시 상호작용을 버리면서 증거 공간을 고정해 시각 토큰의 맥락 압도를 방지한다. 동적 맥락 재구성에 맞춘 상태 마스킹과 신용 할당을 갖춘 GRPO 기반 강화학습 파이프라인으로 VISOR를 훈련한다. ViDoSeek, SlideVQA, MMLongBench 실험에서 VISOR는 장기 지평 시각 추론에서 최고 성능과 우월한 효율을 달성한다.

## Introduction (한글)
LM과 VLM은 빠르게 발전했으나 고정된 학습 데이터에 묶여 환각·지식 공백에 취약하다. RAG는 외부 검색 증거로 출력을 근거화해 이를 완화한다. 실제 문서는 차트·표·그림 등 시각 콘텐츠가 풍부한데 텍스트 추출로는 보존되지 않으므로, Visual RAG(VRAG)는 RAG를 시각 영역으로 확장해 문서 페이지를 렌더링한 이미지를 직접 검색·추론한다. 그러나 기존 VRAG는 단일 라운드 검색의 고정된 retrieve-then-read 파이프라인을 따라 멀티스텝 추론에 필요한 추가 증거를 모으지 못한다. 이를 위해 에이전트형 VRAG가 제안되어 추론과 반복 검색 동작을 교차하며, 최근 연구[26]는 crop-and-zoom 같은 시각 동작으로 행동 공간을 풍부하게 해 검색 이미지의 특정 영역을 선택적으로 확대해 미세 분석을 가능케 한다.

이러한 진전에도 두 병목이 남는다. (1) 시각 증거 희소성: 이미지 간으로는 단서가 여러 페이지에 흩어져 있으나 기존 시스템은 페이지를 고립적으로 처리해 페이지 간 증거 누적 메커니즘이 없고, 이미지 내부로는 관련 정보가 작은 영역(표 셀, 차트 시리즈)에 국한되어 정밀한 crop-and-zoom이 필요한데 평가 메커니즘이 없으면 무관 영역 크롭이나 불필요한 확대로 상호작용 턴을 낭비한다. (2) 장기 지평 검색 표류: 시각 영역에서 증폭되는데, 각 문서 이미지가 수천 시각 토큰을 소비해 맥락 창이 빠르게 포화하고, 이전 증거가 누적 시각 데이터에 묻히며 에이전트 주의가 원 질의에서 멀어진다.

이를 해결하고자 VISOR를 제안한다. 첫째, 시각 증거 희소성 완화를 위해 검색 반복에 걸쳐 질의 관련 관찰을 명시적으로 누적하는 구조화된 Evidence Space를 도입해 고립적 페이지별 처리를 넘어 페이지 간 공동 추론을 가능케 하며, 각 시각 동작의 효용과 결과를 평가해 잡음 크롭을 가지치기하고 검증된 증거만 선택 통합하는 Visual Action Evaluation and Correction을 둔다. 둘째, 검색 표류 극복을 위해 실시간 Evidence Space를 맥락 창 상단에 고정하고 최근 상호작용은 슬라이딩 윈도우로 유지하는 Dynamic Trajectory와, 매 검색 단계에서 원 질의로 재정렬하는 Intent Injection을 둔다. VISOR는 에이전트 루프 구조를 채택해 SFT 후 GRPO 기반 RL로 종단간 학습되며, 보상은 검색 정밀도와 최종 정답성을 함께 최적화한다. ViDoSeek, SlideVQA, MMLongBench에서 VISOR는 작업별 파인튜닝 없이도 기존 에이전트 베이스라인을 능가하고 Qwen2.5-VL-7B 및 3B에서 강한 파인튜닝 베이스라인을 일관되게 능가한다.

## 한 줄 핵심
시각 증거를 누적·정제하는 Evidence Space와 맥락 표류를 막는 동적 궤적·의도 주입을 결합하고 GRPO로 학습해, 장기 지평의 시각 문서 멀티스텝 검색-추론을 효율적으로 수행하는 에이전트형 VRAG.
