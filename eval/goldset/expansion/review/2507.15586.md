# 2507.15586 EviOmni: Learning to Extract Rational Evidence via Reinforcement Learning for Retrieval-Augmented Generation

> section_tag: 생성  ·  survey_source: s1

## 초록 (한글)
검색 증강 생성(RAG)은 대형 언어 모델(LLM)의 정확도를 효과적으로 향상시킨다. 그러나 검색 잡음이 LLM 생성 품질을 크게 저해하여 노이즈 제거 메커니즘 개발이 필요하다. 기존 연구는 깊은 사고 없이 곧바로 증거를 추출하여 핵심 단서를 걸러낼 위험이 있고 일반화에 어려움을 겪는다. 이에 본 논문은 먼저 추론한 뒤 추출하는 방식으로 합리적 증거(rational evidence) 추출을 학습하는 EviOmni를 제안한다. 구체적으로 EviOmni는 증거 추론(evidence reasoning)과 증거 추출(evidence extraction)을 하나의 통합 궤적으로 결합하고, 이어 정보 누출을 막기 위한 지식 토큰 마스킹을 적용하며, 답·길이·형식 측면의 검증 가능한 보상을 사용한 온폴리시 강화학습으로 최적화한다. 다섯 개 벤치마크 데이터셋에 대한 광범위한 실험은 EviOmni의 우수성을 보여주며, 간결하고 고품질의 증거를 제공하고 다운스트림 과제의 정확도를 높이며 전통적 RAG와 에이전트형 RAG 시스템 모두를 지원한다.

## Introduction (한글)
RAG는 오픈 도메인 QA, 사실 검증, 대화 생성 등 지식 집약 과제에서 강한 효과를 보이며 LLM에 널리 쓰인다. 이상적으로 LLM은 순수하게 관련된 내용에 근거해야 정확한 출력과 빠른 추론을 낼 수 있다. 그러나 불완전한 검색 시스템과 잡음 많은 검색 코퍼스 때문에 검색 패시지에는 무관·잡음 스니펫이 많아 LLM 주의를 분산시키고 생성 품질을 떨어뜨린다. 따라서 증거를 추출하고 잡음을 걸러내는 것이 RAG 성능에 필요하다.

최근 연구는 이를 두 부류로 다룬다. (1) 관련 패시지를 검색 목록 상위로 올리는 재순위(reranking), (2) 검색 내용을 일관된 관련 서사로 요약(summarizing). 전자는 재순위 모델 성능에 크게 의존하고 재순위 후 맥락이 깨질 수 있으며, 맥락 이해 부족으로 보통 후자보다 성능이 낮다. 후자는 SFT나 선호 최적화로 LLM을 필터링 모델로 학습하지만, 보통 휴리스틱하게 구성된 학습 데이터(정답이 패시지에 포함되는지 보는 String Inclusion, 단어 중첩을 계산하는 Lexical Overlap)에 의존한다.

효과에도 불구하고 기존 방법은 깊은 사고 없이 곧바로 재순위·요약하여, 검색 내용에 대한 불충분한 맥락 이해로 핵심 단서를 걸러낼 위험이 있다. Figure 1은 단순(vanilla) 증거가 핵심 단서를 누락해 오답으로 이어지는 경우와, 증거 추론이 후속 증거 추출을 안내하여("Passage 2가 관련 있고 ...을 언급한다") 핵심 단서를 보존해 정답으로 이어지는 경우를 비교한다. 이는 먼저 깊이 추론한 뒤 의식적으로 증거를 추출하는 합리적 증거 추출 개발의 동기가 된다.

본 연구는 검증 가능한 보상을 사용한 RL로 RAG용 합리적 증거 추출을 학습하는 EviOmni를 제안한다. 증거 추론(<reason>...</reason>)과 추출(<extract>...</extract>)을 하나의 응답에 통합해 종단간 학습을 가능케 한다. 모델은 여러 응답을 생성해 긍정·부정 경험을 담은 정보성 궤적을 수집한다. 이후 EviOmni는 각 응답에 지식 토큰 마스킹을 적용해 rationale 기반·증거 기반 답(<answer>...</answer>) 생성 시 정보 누출을 막고 각 품질을 평가한다. 마지막으로 답·길이·형식의 세 가지 규칙 기반 검증 가능 보상 함수를 설계해 GRPO로 모델을 최적화한다. 그 결과 EviOmni는 검색 내용을 추론하고 핵심 단서를 간결하면서도 정보성 있는 서사로 통합해 LLM 생성 정확도를 향상시킨다.

## 한 줄 핵심
검색 잡음을 곧바로 걸러내지 않고 "먼저 추론한 뒤 추출"하도록 한 궤적에 통합하고, 검증 가능한 보상의 강화학습으로 학습해 합리적 증거만 뽑아내는 RAG 노이즈 제거 방법.
