# 2508.05748 WebWatcher: Breaking New Frontier of Vision-Language Deep Research Agent

## 초록 (한글)
Deep Research 같은 웹 에이전트는 매우 어려운 정보 탐색 문제를 풀 수 있는 초인적 인지 능력을 보여 왔다. 그러나 대부분의 연구는 주로 텍스트 중심이어서 현실 세계의 시각 정보를 간과한다. 이는 멀티모달 Deep Research를 매우 어렵게 만든다. 그런 에이전트는 텍스트 기반 에이전트에 비해 지각, 논리, 지식, 그리고 더 정교한 도구 사용에서 훨씬 강한 추론 능력을 요구하기 때문이다. 이 한계를 다루기 위해, 향상된 시각-언어 추론 능력을 갖춘 멀티모달 Deep Research 에이전트 WebWatcher를 도입한다. WebWatcher는 효율적인 콜드스타트 학습을 위해 고품질 합성 멀티모달 궤적을 활용하고, 깊은 추론을 위해 다양한 도구를 사용하며, 강화학습으로 일반화를 더욱 강화한다. 멀티모달 에이전트 능력을 더 잘 평가하기 위해, 시각·텍스트 정보를 모두 포함하는 복잡한 정보 검색을 요구하는 BrowseComp 스타일 벤치마크 BrowseComp-VL을 제안한다. 실험 결과 WebWatcher는 네 개의 어려운 VQA 벤치마크에서 독점 기준 모델, RAG 워크플로우, 오픈소스 에이전트를 크게 능가하여 복잡한 멀티모달 정보 탐색 과제 해결의 길을 연다.

## Introduction (한글)
Deep Research 에이전트는 AI의 새로운 프런티어로, LLM이 정적 프롬프트를 넘어 다단계 과제를 계획한다(검색 질의 발행, 문서 읽기, 웹 탐색, 반복 추론으로 답 정교화). 많은 오픈소스 텍스트 전용 웹 에이전트가 BrowseComp, Humanity's Last Exam(HLE) 같은 고난도 벤치마크에서 초인적 능력을 보였다. 그러나 지금까지의 발전은 대부분 텍스트 중심이며, 실제 세계에 편재하는 풍부한 시각 정보를 간과한다. 과학 다이어그램 해석, 그래픽 분석, 시각적으로 풍부한 웹 인터페이스 탐색 같은 과제는 통합된 시각-언어 추론을 요구한다. 독점 에이전트는 진전을 이뤘지만 멀티모달 Deep Research는 거의 미탐구 상태다.

핵심 난제는 현재 멀티모달 Deep Research 에이전트가 특정 시나리오에 한정된 경직된 템플릿 기반 파이프라인에 의존해 실제 연구에 필요한 유연한 추론이 부족하다는 점이다. 한편 많은 VL 에이전트는 주로 시각 도구(OCR, 바운딩 박스 추출, 이미지 크롭, 시각 주석 등)에 의존해 이미지 기반 추론을 하지만, 시각 추론을 깊은 텍스트 이해·교차 모달 추론과 통합하지 못해 복잡한 추론을 요하는 고난도 과제에서 부족하다(Fig.2의 GAIA 사례에서 시각 단독 분석은 실패). 다른 한편 검색 도구만 쓰는 검색 에이전트는 검색 과제에 갇혀, 답이 암묵적이거나 구조화된 상호작용·추가 계산을 요구할 때 실패한다(차트 해석 코드 실행, 단계별 계산, 동적 웹페이지 탐색 등이 필요).

이 격차를 메우기 위해, VL Deep Research 에이전트는 텍스트·시각 양쪽에 강한 추론과 다중 외부 도구의 효과적 사용을 모두 요구한다. 이에 Deep Research 능력을 갖춘 VL 웹 에이전트 WebWatcher를 도입한다.

강한 추론 능력을 위해 고품질 시각 콘텐츠와 복잡한 추론을 결합한 데이터 구성이 필수다. 그러나 현재 VQA 데이터는 대부분 단일 단계 추론의 시각 지각에 집중해 계획 복잡성과 추론 깊이가 부족하다. 이에 깊은 다단계 추론과 전략적 계획에 도움이 되는 학습 데이터 생성 파이프라인을 도입한다. 다양한 웹 소스에 대한 무작위 보행(random walk)으로 실제 지식을 수확해 예측 불가능한 멀티홉 추론 체인을 가진 고난도 QA를 만들고, 핵심 개체를 일반적 서술로 마스킹해 복잡도를 높인다. 그 후 QA-to-VQA 파이프라인으로 멀티모달 VQA 항목으로 변환하고, 다단계 필터링으로 품질을 보장한다.

다중 외부 도구 사용을 위해 Web Image Search, Web Text Search, Webpage Visit, Code Interpreter, 내부 OCR을 통합한다. 고품질 추론 궤적 확보가 난제인데(최근 궤적은 길고 템플릿화되며 다양성이 부족, 다중 도구 조정이 복잡), 이를 위해 행동-관찰 시퀀스에서 프롬프팅으로 추론 궤적을 구성하는 완전 자동 파이프라인을 개발한다. 손수 만든 CoT나 템플릿 rationale과 달리, 본 궤적은 실제 도구 사용 행동에 근거한다. 그 후 합성 고품질 궤적으로 에이전트 모델을 미세조정하고 GRPO로 추가 최적화한다.

마지막으로, BrowseComp의 복잡성을 시각 도메인으로 확장한 도전적 VQA 벤치마크 BrowseComp-VL을 도입한다. 질의는 길고 BrowseComp 스타일이며 개체가 난독화되어 교차 모달 추론, 철저한 정보 탐색, 고수준 계획을 요구한다. WebWatcher는 HLE, LiveVQA, BrowseComp-VL, MMSearch에서 강한 성능을 달성하며 기존 오픈소스 멀티모달 연구 에이전트와 독점 시스템을 일관되게 능가한다.

## 한 줄 핵심
시각과 텍스트를 함께 다루는 멀티모달 Deep Research 웹 에이전트로, 합성 멀티모달 궤적으로 콜드스타트 후 다중 도구 사용을 RL(GRPO)로 일반화하고, 평가용 BrowseComp-VL 벤치마크를 함께 제안.
