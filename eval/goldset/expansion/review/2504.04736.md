# 2504.04736 Synthetic Data Generation & Multi-Step RL for Reasoning & Tool Use

> section_tag: 검색시점;학습형RL  ·  survey_source: s3

## 초록 (한글)
강화학습은 대규모 언어모델(LLM)의 성능 향상에 효과적임이 입증되었다. 그러나 RLHF나 RLAIF 같은 전통적 접근은 문제를 단일 스텝(single-step)으로 다룬다. 초점이 더 복잡한 추론·에이전트 과제로 옮겨가면서, 언어모델은 해답을 내놓기 전에 여러 단계의 텍스트 생성, 추론, 환경 상호작용을 수행해야 한다. 본 연구는 다단계(multi-step) 최적화 시나리오를 겨냥한 합성 데이터 생성 및 RL 방법론을 제안한다. 이 접근은 SWiRL(Step-Wise Reinforcement Learning)이라 불리며, 다단계 추론 및 도구 사용 데이터를 반복적으로 생성하고 그 데이터로부터 학습한다. 각 다단계 궤적(trajectory)을 원래 모델의 각 행동(action)에 해당하는 여러 하위 궤적(sub-trajectory)으로 쪼개는 단순한 단계별 분해를 사용한다. 그런 다음 이 하위 궤적들에 합성 데이터 필터링과 RL 최적화를 적용한다. 다단계 도구 사용, 질의응답, 수학 추론 과제들에서 SWiRL을 평가한 결과, GSM8K, HotPotQA, CofCA, MuSiQue, BeerQA에서 각각 상대 정확도 21.5%, 12.3%, 14.8%, 11.1%, 15.3% 만큼 기준선을 능가했다. 흥미롭게도 이 접근은 과제 간 일반화를 보인다. 예컨대 HotPotQA(텍스트 QA)만으로 학습해도 GSM8K(수학 데이터셋)의 zero-shot 성능이 상대적으로 16.9% 향상되었다.

## Introduction (한글)
LLM은 자연어 처리에서 뛰어난 능력을 보여왔지만, 다중 홉 QA, 수학 문제 풀이, 코딩 등 여러 단계에 걸친 추론과 도구 사용이 필요한 복잡한 질의에는 종종 어려움을 겪는다. RLHF, RLAIF, RLEF 같은 전통적 RL 접근은 단일 스텝 최적화에 집중해, 다단계 과제의 어려움을 거의 다루지 못했다. 많은 실제 문제는 서로 연관된 일련의 행동을 요구한다. 예를 들어 어려운 질문에 답할 때, 모델은 어떤 정보를 찾을지뿐 아니라 언제 탐색을 멈추고 발견한 것을 종합할지도 결정해야 한다. 다단계 추론은 중간 단계의 오류가 최종 결과의 오류로 이어지는 복합적 난제를 만들기 때문에, 전체 행동 사슬에 걸쳐 정확성을 유지하거나 오류로부터 효과적으로 회복하는 능력이 중요하다.

이를 해결하기 위해 오프라인 다단계 최적화 기법인 SWiRL을 제시한다. 모델이 검색 엔진이나 계산기 같은 도구에 접근할 수 있고, 질문에 답하기 위해 필요에 따라 일련의 도구 호출을 실행할 수 있는 환경을 가정한다. 목표는 복잡한 문제를 더 다루기 쉬운 하위 과제들의 연쇄로 분해하는 법, 언제 도구를 호출하고 어떻게 호출을 구성할지, 언제 그 결과를 사용해 답하고 어떻게 발견을 종합할지를 모델에게 가르치는 것이다. 두 단계 접근으로, 먼저 다단계 합성 데이터를 생성한 뒤 단계별 RL로 학습한다. 이 오프라인 방식은 병렬 호출로 대량의 데이터를 빠르게 생성할 수 있고, 고정 데이터셋 덕에 재현성이 높다.

합성 데이터 생성을 위해 오픈소스 LLM(Gemma 2)에 관련 도구를 제공하고, 반복적으로 다단계 궤적을 생성하게 한다. 각 단계에서 모델은 사고 사슬을 생성하고 도구를 호출하거나 최종 답을 낼 수 있다(이를 행동이라 부른다). k개 행동을 가진 궤적을 k개 하위 궤적으로 변환하고, 각 하위 궤적의 맥락에서 각 행동을 평가하는 생성형 보상 모델을 사용해 단계별 RL을 적용한다. DeepSeek-R1, Llama-3 같은 기존 RL 미세조정과 달리, 최종 성능만을 최적화하지 않고 골든 라벨도 쓰지 않으며, 이전 단계가 주어졌을 때 각 단계의 합리성을 최적화함으로써 최종 성능까지 향상시킨다. 기여로는 SWiRL 제안, 데이터셋 및 이질적 과제 간 일반화·전이 입증, 합성 데이터 필터링 전략 분석(결과 기반이 아닌 단계별 필터링이 가장 효과적), 데이터·모델 크기 영향 분석 등이 있다. ## 한 줄 핵심
다단계 추론·도구 사용 궤적을 하위 궤적으로 쪼개 각 단계마다 보상을 주는 오프라인 단계별 강화학습(SWiRL)으로, 골든 라벨 없이도 다중 홉 QA와 수학 추론을 향상시키는 방법.
