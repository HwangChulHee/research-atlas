# 2507.15061 WebShaper: Agentically Data Synthesizing via Information-Seeking Formalization

> section_tag: 학습형RL  ·  survey_source: s2

## 초록 (한글)
대형 언어 모델(LLM) 기반 에이전트의 등장은 웹 기반 정보 탐색(IS) 능력을 통해 복잡하고 개방형인 과제 해결을 가능케 하며 인공지능을 혁신해 왔다. 그러나 고품질 학습 데이터의 부족이 IS 에이전트 개발을 제한해 왔다. 기존 데이터 합성 접근은 보통 웹 데이터를 먼저 수집한 뒤 검색 결과를 바탕으로 질문을 생성하는 정보 주도(information-driven) 패러다임을 채택하는데, 이는 정보 구조와 추론 구조 사이, 그리고 질문과 그에 대응하는 답 사이의 불일치를 초래할 수 있다. 이를 완화하기 위해, 집합론적 구성으로 IS 과제를 체계적으로 형식화하는 형식화 주도(formalization-driven) IS 데이터 합성 프레임워크 WebShaper를 제안한다. 형식화의 핵심은 지식 투영(Knowledge Projections, KP) 개념으로, KP 연산의 조합을 통해 추론 구조를 정밀하게 제어할 수 있다. 합성 과정에서 시드 과제를 만든 뒤 다단계 확장 절차를 사용한다. 각 단계마다 에이전트형 Expander가 형식화에 기반해 검색·검증 도구로 현재 형식 질문을 더 복잡하게 확장한다. 합성된 데이터셋으로 모델을 학습한 결과, WebShaper는 GAIA와 WebWalkerQA 벤치마크에서 오픈소스 IS 에이전트 중 최첨단 성능을 달성한다.

## Introduction (한글)
LLM 기반 언어 에이전트의 등장은 여러 도메인에서 변혁적 해법을 가능케 한 패러다임 전환적 진전이다. 정보 탐색(IS)은 언어 에이전트 인지 자율성의 핵심 구성 요소로, 개방형 과제 적응력의 기반이자 OpenAI Deep Research, Gemini, Perplexity 같은 상용 시스템의 동력이다.

현재 에이전트 시스템은 보통 (1) 과제 특화 QA 궤적 구성, (2) SFT로 기초 기술 습득, (3) 온폴리시 RL로 전략적 의사결정 일반화의 파이프라인을 따른다. IS 에이전트 개발 전체는 고품질 IS 학습 데이터에서 출발하며 그 효과에 의존하지만, 복잡성 때문에 크라우드소싱으로 구축하기 어렵고 희소하다. 따라서 잘 설계된 에이전트 파이프라인을 통한 학습 데이터 구성이 핵심이 된다.

기존 IS 데이터 합성법은 보통 온라인에서 정보를 자유롭게 사전 검색한 뒤 LLM으로 수집 내용에서 질문을 생성한다(정보 주도). 이들은 수집 정보를 구조화 포맷으로 정리한 뒤 자연어 질문을 만들며, 핵심 목표는 정보 구조를 질문의 추론 구조로 사상하는 것이다. 대표적으로 WebDancer와 TaskCraft는 선형 정보 체인을 생성하고, 다른 연구들은 웹 링크로 연결된 그래프나 개체 공참조 네트워크를 구성한다. 그러나 정보 주도 접근은 두 가지 한계가 있다. 첫째, LLM이 정보 구조를 충분히 이해하지 못해 추론 구조가 불일치하거나 생성된 질문에 대한 답이 틀릴 수 있다. 둘째, 무질서한 정보 검색으로 과도한 데이터 처리와 중복·동질적 정보 구조 수집이 발생해 다양성과 지식 커버리지가 줄어든다.

이를 극복하기 위해 형식화 주도 IS 데이터 합성 패러다임 WebShaper를 제안한다. 기존과 달리 IS 과제를 먼저 형식화하고 이 형식화를 통해 데이터 합성을 체계적으로 안내한다. 생성 중 정보 수집은 형식 과제 요구로 명시적으로 제어된다. 이 프레임워크의 장점은 (1) 더 넓은 과제 커버리지, (2) 과제 제어 가능성, (3) 구조·답 일관성이다.

WebShaper 데이터셋을 구축해 IS 에이전트의 학습 데이터로 삼는다. 핵심은 IS 과제 형식화로, 수집 복잡도와 추론 경로를 제어 가능하게 과제 인스턴스를 원리적으로 생성한다. 수학 증명의 Lean 4나 지식 중심 QA의 명제 논리처럼 사전 형식화가 존재하는 분야와 달리, 정보 탐색에는 확립된 형식화가 없다. 저자들은 집합론에 기반해 이를 최초로 도출했다고 밝히며, IS를 기본 단위인 지식 투영(KP)의 조합으로부터 과제가 체계적으로 도출되는 통합 문제 공간으로 다룬다. 시드 과제를 구성한 뒤 형식 프레임워크에 근거한 다단계 확장을 수행하며, 전용 에이전트형 Expander 모듈이 KP 표현으로 과제 요구를 해석한다. Expander는 (1) 형식 요구에 따른 인터넷 기반 지식 수집, (2) 새 형식 문제 구성·검증, (3) 최종 질문 생성의 세 기능을 자율 수행한다.

실험으로 WebShaper 데이터셋의 유효성을 검증한다. 기존 학습 데이터셋과의 비교는 효과를 보여주며, GAIA·WebWalkerQA 벤치마크에서 모든 오픈소스 IS 에이전트 중 최고 성능을 달성한다.

## 한 줄 핵심
정보를 먼저 모아 질문을 짓는 대신, 집합론(지식 투영 KP) 기반으로 정보 탐색 과제를 먼저 형식화하고 에이전트형 Expander로 점진 확장해 일관성 높은 학습 데이터를 합성하는 방법.
