# 2501.02727 Tree-based RAG-Agent Recommendation System: A Case Study in Medical Test Data

> section_tag: 추론워크플로우  ·  survey_source: s1

## 초록 (한글)
우리는 HiRMed(계층적 RAG 강화 의료 검사 추천)를 제시한다. 이는 검색증강생성(RAG)을 활용한 지능형 의료 검사 추천을 위한 새로운 트리 구조 추천 시스템이다. 전통적인 벡터 유사도 기반 접근법과 달리, 우리 시스템은 각 트리 노드에서 전문화된 RAG 과정을 통해 의료 추론을 수행한다. 초기 증상을 가진 루트 노드에서 시작하여, 시스템은 단계별 의료 분석을 수행해 잠재적 기저 질환과 그에 대응하는 진단 요건을 식별한다. 각 단계에서 단순 매칭 대신, RAG 강화 노드는 검색된 의료 지식을 분석하여 증상-질환 관계를 이해하고 가장 적합한 진단 경로를 결정한다. 시스템은 긴급도와 진단 불확실성 등을 고려하여 의료 추론 결과에 따라 추천 전략을 동적으로 조정한다. 실험 결과 우리 접근법은 커버리지율, 정확도, 누락률 면에서 기존 검색 기반 방법보다 우수한 성능을 달성한다. 본 연구는 전통적 트리 기반 검색 구조에 의료 추론 능력을 도입함으로써 의료 검사 추천 분야의 중요한 진전을 나타낸다.

## Introduction (한글)
의료 검사 추천은 현대 의료 시스템에서 진단 정확도, 치료 결과, 자원 활용에 직접 영향을 미치는 핵심 역할을 한다. 규칙 기반 시스템이나 유사도 기반 검색 같은 전통적 접근법은 널리 쓰여왔으나, 의료 진단의 미묘하고 맥락 의존적인 본질을 포착하는 데 종종 부족하다.

최근 LLM과 RAG의 발전은 다양한 의료 응용에서 유망한 결과를 보였으나, 의료 검사 추천에 직접 적용하기에는 몇 가지 한계가 있다. 의료 진단 과정을 반영하는 계층적 추론 구조의 부재, 진단 단계별 도메인 특화 의료 지식 통합 부족, 추천 과정 전반의 추론 이력 유지·활용 능력 제한, 포괄적 커버리지와 표적 특이성 사이의 균형 문제 등이다.

이를 해결하기 위해 우리는 HiRMed를 제안한다. 세 가지 핵심 혁신으로 구별된다. 첫째, 계층적 RAG 아키텍처로 각 노드가 전문화된 RAG 과정을 포함하는 트리 구조를 구현하여, 일반 증상 평가부터 특정 검사 선택까지 여러 수준의 의료 추론을 통한 추천의 점진적 정련을 가능케 한다. 둘째, 동적 지식 통합으로 이중 계층 지식베이스(루트 수준의 광범위 의료 이해용과 부서별 전문 진단 고려용)를 갖춘다. 셋째, 메모리 강화 추론으로 노드 간 추론 이력을 유지하는 정교한 메모리 메커니즘을 도입하여 일관되고 포괄적인 진단 경로를 가능케 하면서 중복·충돌 추천을 회피한다.

(관련 연구 일부 요지만 보존: 트리 기반 추천 시스템, RAG, 의료 진단 응용을 검토하며, 기존 트리 기반 시스템은 정적 규칙 기반 접근에 의존해 동적 의료 데이터에 대한 적응성이 제한적이라고 지적한다.)

## 한 줄 핵심
각 트리 노드에서 RAG 추론을 수행하는 계층적 트리 구조로, 증상에서 출발해 단계적으로 적합한 의료 검사를 추천하는 시스템이다.
