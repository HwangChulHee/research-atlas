# 2510.05096 Paper2Video: Automatic Video Generation from Scientific Papers

> section_tag: 생성  ·  survey_source: s2

## 초록 (한글)
학술 발표 영상은 연구 소통의 핵심 매체가 되었지만, 제작은 여전히 노동집약적이어서 2~10분 분량의 짧은 영상 하나에 슬라이드 설계, 녹화, 편집에 수 시간이 든다. 자연 영상과 달리 발표 영상 생성은 독특한 도전을 수반한다: 연구 논문에서 오는 긴 문맥 입력, 밀집된 멀티모달 정보(텍스트, 그림, 표), 그리고 슬라이드·자막·음성·발표자 영상 등 정렬된 여러 채널을 조율해야 하는 필요성. 이를 해결하기 위해 우리는 Paper2Video를 소개한다. 이는 101편의 연구 논문을 저자 제작 발표 영상, 슬라이드, 발표자 메타데이터와 짝지은 최초의 벤치마크다. 또한 영상이 청중에게 논문 정보를 얼마나 잘 전달하고 연구 영향력에 어떻게 기여하는지 측정하기 위해 네 가지 맞춤형 평가 지표(Meta Similarity, PresentArena, PresentQuiz, IP Memory)를 설계했다. 이 기반 위에 우리는 학술 발표 영상 생성을 위한 최초의 다중 에이전트 프레임워크 PaperTalker를 제안한다. PaperTalker는 새로운 효과적 Tree Search Visual Choice를 통한 레이아웃 정제와 슬라이드 생성, 커서 그라운딩, 자막, 음성 합성, 토킹헤드 렌더링을 통합하면서 슬라이드별 생성을 병렬화해 효율을 높인다. Paper2Video 실험에서 우리 방법이 만든 발표 영상이 기존 베이스라인보다 더 충실하고 정보가 풍부함을 보여, 자동화되고 즉시 사용 가능한 학술 영상 생성을 향한 실용적 진전을 이뤘다.

## Introduction (한글)
학술 발표 영상은 연구 소통에 널리 쓰이며 많은 학회가 제출 필수 자료로 요구한다. 그러나 수작업 제작은 슬라이드 설계, 자막 작성, 슬라이드별 녹화, 세심한 편집이 필요해 노동집약적이며 영상 하나에 평균 수 시간이 든다. 슬라이드/포스터 생성과 여타 AI4Research 과제에 대한 선행 연구가 있지만, 자동 학술 발표 영상 생성은 그들의 상위 문제(superproblem)로 더 실용적이면서도 어려운 방향이다. 자연 영상 생성과 달리 발표 영상은 다감각 통합, 다중 그림 조건화, 높은 텍스트 밀도를 보여 현 자연 영상 생성 모델의 한계를 드러낸다. 구체적 도전: a. 밀집 텍스트와 다수 그림·표를 담은 긴 문맥 논문에서 출발, b. 슬라이드 생성·자막·TTS·커서 제어·토킹헤드 등 정렬된 다중 채널 조율, c. 평가 지표 부재(좋은 발표 영상이 무엇인지, 특히 지식 전달과 청중 접근성 측면). 최첨단 종단간 비디오-오디오 모델 Veo3조차 영상 길이, 밀집 텍스트 선명도, 멀티모달 장문서 조건화에서 한계가 있다.

포괄적 평가를 위해 Paper2Video 벤치마크(101편 논문-저자 녹화 영상, 원본 슬라이드, 발표자 신원 메타데이터)를 제시하고 다차원 평가 지표를 개발했다: (i) Meta Similarity — VLM으로 생성 슬라이드·자막과 사람 제작본의 정렬 평가, (ii) PresentArena — VideoLLM을 대리 청중으로 양방향 쌍별 비교, (iii) PresentQuiz — VideoLLM을 청중으로 논문 유래 질문에 답하게 함, (iv) IP Memory — 영상 시청 후 청중이 저자·연구를 얼마나 잘 연관시키는지 측정.

또한 PaperTalker를 제안한다. 이는 논문과 발표자 신원으로부터 학술 발표 영상을 생성하는 최초의 다중 에이전트 프레임워크다. 모듈: (i) 슬라이드 생성 — 템플릿 pptx/XML 대신 LaTeX 코드로 슬라이드를 처음부터 생성하고, 최첨단 Coder로 코드 생성 + 집중 디버깅 전략(관련 행 피드백으로 범위 좁혀 컴파일 오류 해결). 미세 수치 조정에 둔감한 LLM 문제를 위해 Tree Search Visual Choice 제안 — 파라미터 변형으로 여러 분기를 만들어 하나의 그림으로 합치고 VLM이 최적 분기 선택(그림·폰트 크기 등 레이아웃 개선). (ii) 자막·커서 그라운딩 — 슬라이드 기반 문장별 자막·커서 프롬프트 생성, Computer-use 그라운딩 모델과 WhisperX로 시공간 정렬. (iii) 음성 합성·토킹헤드 렌더링 — TTS로 개인화 음성 합성, 토킹헤드 영상 생성. 슬라이드 간 독립성을 활용해 슬라이드별 생성을 병렬화해 6배 이상 속도 향상.

## 한 줄 핵심
연구 논문을 입력하면 슬라이드·자막·음성·커서·토킹헤드까지 갖춘 학술 발표 영상을 자동 생성하는 다중 에이전트 시스템(PaperTalker)과 이를 평가할 101편 벤치마크(Paper2Video).
