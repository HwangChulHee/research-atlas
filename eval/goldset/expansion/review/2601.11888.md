# 2601.11888 Agentic-R: Learning to Retrieve for Agentic Search

## 초록 (한글)
에이전트 검색은 에이전트가 다단계 추론과 필요 시점의 검색(on-demand retrieval)을 교차해 복잡한 질문을 푸는 강력한 패러다임으로 최근 부상했다. 그 성공에도 불구하고 에이전트 검색을 위한 검색기(retriever)를 어떻게 설계할지는 거의 탐구되지 않았다. 기존 검색 에이전트는 보통 유사도 기반 검색기에 의존하나, 유사한 구절이 항상 최종 답 생성에 유용한 것은 아니다. 본 논문은 에이전트 검색에 맞춘 새로운 검색기 학습 프레임워크를 제안한다. 지역적(local) 구절 효용만 보는 단일 회전 RAG용 검색기와 달리, 다회전 에이전트 검색에서 구절 효용을 측정하기 위해 지역적 질의-구절 관련성과 전역적 답 정확성을 모두 사용한다. 나아가 검색 에이전트와 검색기를 양방향·반복적으로 최적화하는 반복 학습 전략을 도입한다. 고정 질문으로 한 번만 학습하는 RAG 검색기와 달리, 본 검색기는 에이전트가 진화시키는 고품질 질의로 지속적으로 개선된다. 일곱 개 단일 홉·다중 홉 QA 벤치마크 실험에서 Agentic-R로 명명된 본 검색기가 다양한 검색 에이전트에 걸쳐 강한 기준선들을 일관되게 능가함을 보인다.

## Introduction (한글)
RAG는 외부 정보를 검색해 LLM의 지식 한계를 보완하는 널리 채택된 접근이 되었다. 최근 대형 추론 모델의 발전은 에이전트 검색이라는 새 패러다임을 낳았다. 이는 전통적 단일 회전 검색을 다단계 "사고 중 검색(search-during-think)" 과정으로 확장해, 에이전트가 복잡한 질문을 하위 질의 시퀀스로 분해하고 여러 턴에 걸쳐 추론과 검색을 교차하며 점진적으로 증거를 모은 뒤 최종 답을 낸다. 추론과 검색을 통합해 에이전트 검색은 도전적 NLP 과제에서 전통적 단일 회전 RAG보다 우수하다.

이 진전에도 대부분의 에이전트 검색 연구는 더 강력한 에이전트 설계에 집중하고, 결정적 구성요소인 검색기 최적화에는 관심을 덜 둔다. 검색된 구절 품질이 학습과 추론 성능에 직접 영향을 주므로 이는 중대한 누락이다. 현재 대부분은 의미 유사도에 의존하는 기성 검색기(E5, BGE)를 쓰는데, 높은 의미 유사도가 답 생성에 유용함을 보장하지는 않는다.

검색을 다운스트림 생성에 맞추기 위해 선행 연구는 구절 효용 모델링과 효용 최적화 검색기를 탐구했다. 이들은 보통 검색된 구절을 생성기에 넣어 생성 우도나 과제별 지표 같은 다운스트림 성능을 측정해 효용을 추정한다. 단일 회전 RAG엔 효과적이나, 다회전 에이전트 검색으로 확장하면 큰 난제가 있다. 첫째, 단일 회전 RAG의 효용 모델링은 데이터셋의 정답에 의존하지만, 에이전트 검색의 중간 질의에는 정답이 없어 효용 평가가 어렵다. 둘째, 에이전트 검색의 구절 효용은 지역적 관련성을 넘어선다. 현재 하위 질의에 관련된 구절이 오도성 정보를 담아 후속 추론을 잘못된 방향으로 이끌 수 있어, 지역적 관련성만으로는 불충분하다. 또한 기존 RAG 검색기 학습은 고정 질의와 고정 생성기로 한 번만 학습하는 일방향 패러다임을 채택하는데, 에이전트 검색에선 학습 질의가 에이전트 자신이 생성하므로 부적절하다. 검색기가 개선되면 에이전트가 더 강한 검색기와 상호작용해 RL로 향상되고, 그 에이전트가 더 좋은 질의 궤적을 만들어 다시 검색기를 개선하므로, 최적화는 양방향·반복 과정으로 정식화되어야 한다. 본 논문은 에이전트 검색 전용 최초의 검색기 학습 프레임워크를 제안한다. ## 한 줄 핵심
검색 에이전트의 약한 고리인 검색기를, "지역적 관련성 + 최종 답 정확성"으로 구절 효용을 재정의하고 에이전트와 검색기를 번갈아 키우는 반복 학습으로 개선한 전용 검색기(Agentic-R).
