# 2510.10095 CardRewriter: Leveraging Knowledge Cards for Long-Tail Query Rewriting on Short-Video Platforms

> section_tag: 질의계획  ·  survey_source: s2

## 초록 (한글)
숏폼 영상 플랫폼은 빠르게 차세대 정보 검색 시스템이 되어, 사용자가 질의를 만들어 원하는 영상에 접근한다. 그러나 사용자 질의, 특히 롱테일(long-tail) 질의는 철자 오류, 불완전한 표현, 모호한 의도를 자주 겪어 사용자 기대와 검색 결과 간 불일치를 초래한다. 대형 언어 모델(LLM)은 전자상거래에서 롱테일 질의 재작성에 성공을 보였지만, 숏폼 영상·라이브 스트림·마이크로 드라마·사용자 소셜 네트워크 같은 독점 콘텐츠가 훈련 분포 밖에 있는 숏폼 플랫폼에서는 어려움을 겪는다. 이를 해결하기 위해 우리는 도메인 특화 지식을 통합해 롱테일 질의 재작성을 향상하는 LLM 기반 프레임워크 CardRewriter를 소개한다. 각 질의에 대해 우리 방법은 질의 관련 다중 출처 지식을 집계하고 이를 정보가 풍부하고 질의 관련성 높은 "지식 카드(knowledge card)"로 요약한다. 이 카드는 LLM이 사용자 의도를 더 잘 포착하고 더 효과적인 질의 재작성을 생성하도록 안내한다. 우리는 CardRewriter를 2단계 훈련 파이프라인(지도 미세조정 후 그룹 상대 정책 최적화 GRPO)으로 최적화하며, 질의 관련성과 검색 효과를 균형 잡는 맞춤 보상 시스템을 사용한다. 오프라인 실험에서 CardRewriter는 독점 콘텐츠 대상 질의의 재작성 품질을 크게 개선했다. 온라인 A/B 테스트는 롱뷰율(LVR)과 클릭률(CTR)의 유의미한 향상, 능동적 질의 재구성률(IQRR)의 현저한 감소를 추가로 확인했다. 2025년 9월부터 CardRewriter는 중국 최대 숏폼 플랫폼 중 하나인 콰이쇼우(Kuaishou)에 배포되어 매일 수억 사용자에게 서비스되고 있다.

## Introduction (한글)
최근 숏폼 플랫폼(콰이쇼우, 틱톡, 샤오훙슈 등)은 일상에 보편화되었고, 점점 검색 엔진 역할도 하여 사용자가 질의를 던지고 정보 요구에 맞춘 순위화된 영상 결과를 받는다. 그러나 표현·의도의 다양성으로 인해 철자 오류, 누락된 용어, 의미 모호성 같은 문제가 특히 롱테일 질의에서 발생한다. 이런 질의는 만족스럽지 않은 결과를 내어 사용자가 반복 재작성하거나 검색을 포기하게 만들므로, 이 문제 해결이 사용자 만족에 핵심적이다.

질의 재작성은 직관적이고 널리 쓰이는 해법으로, 언어 오류 교정, 누락 키워드 보충, 질의 스타일을 콘텐츠에 맞추는 일을 한다. 기존 방법은 두 패러다임으로 나뉜다: (I) 임베딩 기반 — 의미적으로 유사한 질의를 검색해 원 질의를 풍부하게 하나, 의미 매칭이 드문 롱테일 질의에서 한계. (II) 생성 기반 — 롱테일 질의를 직접 재작성해 의미 명료성·완전성을 높임. 최근 LLM 기반 재작성이 강화학습으로 사용자 질의를 플랫폼 특유 표현 스타일에 정렬하며 지배적 접근이 되었다.

LLM 기반 재작성은 전자상거래의 롱테일 질의에서 강한 성능을 보였지만 숏폼 플랫폼 확장은 여전히 어렵다. 이런 플랫폼은 숏폼 영상·라이브·마이크로 드라마·소셜 네트워크 등 LLM이 사전학습 중 접하지 못한 방대한 독점 콘텐츠를 담고 있어 재작성 실패로 이어진다. (예: 사용자가 크리에이터 "Coco-Cola"의 영상 시리즈를 찾으려 "Coca-Cola"로 오타를 내면, LLM이 음료 브랜드로 잘못 연관시켜 "창의적 광고"를 덧붙여 전혀 무관한 결과를 반환.)

이를 위해 우리는 CardRewriter를 제안한다. 이는 플랫폼 특화 지식을 활용해 LLM 기반 질의 재작성 품질을 높이는 검색 증강 프레임워크다. 입력 질의에 대해 먼저 숏폼 플랫폼에서 다중 출처 지식을 검색한다 — 원 질의 및 유사 고공급 질의 집합 각각으로 검색한 상위 k개 영상의 멀티모달 콘텐츠와 관련 오픈도메인 문서. 이 수집 지식을 입력 질의에 맞춘 간결한 지식 카드로 요약하고, 이를 재작성 과정에 통합해 LLM이 검색 맥락을 더 잘 이해하고 효과적 재작성을 내도록 안내한다. 고품질 카드·재작성 보장을 위해 SFT와 GRPO를 결합한 2단계 훈련 패러다임을 채택하고, 질의 관련성과 검색 효과를 균형 잡는 맞춤 보상을 사용한다.

## 한 줄 핵심
숏폼 플랫폼 독점 콘텐츠에 대한 사용자의 롱테일·오타 질의를, 다중 출처 지식을 요약한 "지식 카드"로 LLM을 안내해 정확하게 재작성하는 (SFT+GRPO 학습) 실서비스 프레임워크.
