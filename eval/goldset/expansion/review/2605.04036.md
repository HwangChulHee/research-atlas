# 2605.04036 OpenSeeker-v2: Pushing the Limits of Search Agents with Informative and High-Difficulty Trajectories

> section_tag: 검색시점  ·  survey_source: s3

## 초록 (한글)
딥 서치 능력은 최전선 LLM 에이전트의 필수 역량이 되었으나, 그 개발은 여전히 산업계 거대 기업이 지배한다. 전형적 산업 레시피는 사전학습, 지속 사전학습(CPT), 지도 파인튜닝(SFT), 강화학습(RL)에 걸친 고자원 파이프라인이다. 본 보고서는 정보량 많고 고난도인 궤적을 공급하면 단순한 SFT 접근만으로도 최전선 검색 에이전트 훈련에 놀랍도록 강력할 수 있음을 보인다. 세 가지 단순한 데이터 합성 수정—더 풍부한 탐색을 위한 지식 그래프 크기 확장, 더 넓은 기능을 위한 도구 집합 확장, 엄격한 저단계 필터링—을 도입해 더 강한 베이스라인을 세운다. 단 10.6k 데이터로 훈련한 OpenSeeker-v2는 4개 벤치마크(ReAct 패러다임 30B급 에이전트)에서 SOTA를 달성한다: BrowseComp 46.0%, BrowseComp-ZH 58.1%, Humanity's Last Exam 34.6%, xbench 78.0%로, 무거운 CPT+SFT+RL 파이프라인의 Tongyi DeepResearch(각각 43.4%, 46.7%, 32.9%, 75.0%)마저 능가한다. 특히 OpenSeeker-v2는 해당 모델 규모·패러다임에서 순수 학계 팀이 SFT만으로 개발한 최초의 SOTA 검색 에이전트다. 모델 가중치를 오픈소스로 공개한다.

## Introduction (한글)
정보 폭발 시대에 딥 서치는 최전선 LLM 에이전트의 필수 역량이 됐다. 그러나 고성능 에이전트 개발은 오랫동안 자금이 풍부한 기업들의 "닫힌 게임"이었다. SOTA 달성을 위한 전형적 산업 레시피는 대규모 코퍼스의 지속 사전학습(CPT), 이어 SFT, 그리고 복잡한 RL 단계로 매우 자원 집약적이며, 막대한 연산과 독점 데이터 파이프라인 의존이 학계·오픈소스 커뮤니티의 혁신을 근본적으로 가로막아 왔다.

저자들은 이 복잡한 다단계 훈련 파이프라인 의존에 도전한다. 초기 탐색 OpenSeeker를 기반으로, 초점을 훈련 궤적 자체의 품질로 완전히 옮겨 "단순한 SFT 접근만으로 검색 에이전트의 한계를 밀어붙여 무거운 산업 파이프라인 성능에 필적할 수 있는가?"라고 묻는다.

본 보고서는 OpenSeeker-v2를 소개하며, 고난도·풍부한 데이터를 공급하면 단순 SFT로 충분히 강력함을 입증한다. 데이터 합성 파이프라인에 세 가지 단순하지만 효과적인 수정을 도입한다. (1) 더 풍부한 탐색을 위한 그래프 크기 확장: 데이터 생성 시 위상 그래프 크기를 크게 늘려 더 풍부·다양한 출처 정보를 맥락에 주입하고, 구조적으로 깊은 멀티홉 탐색을 요구하는 매우 복잡한 과제를 합성한다. (2) 더 넓은 기능을 위한 도구 집합 확장: 사용 가능 도구 수를 늘려 더 다재다능한 전략을 학습하고 다양한 질의를 처리하게 한다. (3) 엄격한 저단계 필터링: 너무 적은 도구 호출 단계로 풀리는 궤적을 걸러내, 최소 난이도 하한을 보장하고 장기 지평의 지속적 추론·정보 탐색을 강제한다. 이 전략으로 단 10k의 고난도 궤적을 정제하고, 30B 모델을 단일 SFT 실행으로 훈련해 4개 대표 벤치마크에서 새 SOTA를 달성한다(Tongyi DeepResearch의 무거운 파이프라인을 결정적으로 능가). OpenSeeker-v2는 해당 규모·패러다임(ReAct)에서 순수 학계 팀이 SFT만으로 개발한 최초의 SOTA 검색 에이전트이며, 재현 가능한 베이스라인으로 모델 가중치를 완전 공개한다.

## 한 줄 핵심
복잡한 CPT+SFT+RL 대신 고난도·정보량 많은 궤적을 합성(그래프 확장·도구 확장·저단계 필터링)해 단순 SFT만으로 30B 딥 서치 에이전트를 SOTA로 끌어올린 기술 보고서.
