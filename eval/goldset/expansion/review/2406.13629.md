# 2406.13629 InstructRAG: Instructing Retrieval-Augmented Generation via Self-Synthesized Rationales

> section_tag: 정보필터링  ·  survey_source: s2

## 초록 (한글)
검색 증강 생성(RAG)은 언어 모델(LM)의 정확성과 사실성을 높일 유망한 잠재력을 보였다. 그러나 불완전한 검색기나 노이즈가 많은 코퍼스는 검색 내용에 오도하거나 심지어 잘못된 정보를 들여와 생성 품질에 큰 도전을 준다. 기존 RAG 방법은 보통 노이즈가 있더라도 최종 답을 직접 예측함으로써 이 문제를 다루는데, 이는 해석·검증이 어려운 암묵적 노이즈 제거(denoising) 과정을 낳는다. 한편 명시적 노이즈 제거 감독(supervision)을 얻으려면 상당한 인간 노력이 들어 비용이 크다. 본 연구는 LM이 자기-합성 근거(self-synthesized rationales)를 통해 노이즈 제거 과정을 명시적으로 학습하는 INSTRUCTRAG를 제안한다. 먼저 LM에게 정답이 검색 문서로부터 어떻게 도출되는지 설명하도록 지시한다. 그런 다음 이 근거를 명시적 노이즈 제거를 위한 인컨텍스트 학습의 시연(demonstration)으로 쓰거나, 모델 훈련을 위한 지도 미세조정(SFT) 데이터로 쓴다. 표준 RAG 대비 INSTRUCTRAG는 추가 감독이 필요 없고, 예측 답의 검증이 쉬우며, 생성 정확도를 효과적으로 높인다. 실험 결과 학습 없는(training-free) 및 학습 가능한(trainable) 두 시나리오 모두에서 기존 RAG 방법을 일관되게 능가해, 다섯 개 지식 집약 벤치마크 평균에서 최고 기준선 대비 상대 8.3% 향상을 달성했다. 또한 검색 문서 수 증가에 잘 확장되고, 도메인 밖 데이터셋에서도 강건한 노이즈 제거 능력을 보여 강한 일반화를 입증했다.

## Introduction (한글)
LLM은 뛰어난 텍스트 생성 능력을 보이나, 특히 최신 정보나 사전학습 코퍼스에 충분히 반영되지 않은 도메인 밖 지식이 필요할 때 사실적으로 틀린 내용을 생성하곤 한다. 이는 사실성이 중요한 고위험 도메인에서의 신뢰성 있는 배치를 가로막는다. 이에 RAG가 외부 소스의 최신·전문 지식을 활용해 지식 집약 과제의 생성 정확도를 높이기 위해 도입됐다. 그러나 완벽한 검색이 없고 검색 코퍼스에 노이즈가 존재해, 검색 내용은 보통 무관하거나 잘못된 정보와 섞여 거의 모든 RAG 시스템의 오랜 난제가 된다. 전형적 vanilla RAG는 노이즈 입력에도 정답을 직접 예측하도록 LM을 훈련해 이 문제를 암묵적으로 다룬다. (이하 본문은 head6000으로 잘려 INSTRUCTRAG의 구체 방법 후반부는 본문에 미포함 — 압축됨.) Figure 1에 따르면 vanilla RAG는 질의와 노이즈 검색 문서로부터 명시적 노이즈 제거나 도출 설명 없이 답을 직접 예측하는 반면, INSTRUCTRAG는 검색 문서를 명시적으로 노이즈 제거하고 예측 답을 정당화하는 근거를 생성해 정확도와 신뢰성을 함께 높인다.

## 한 줄 핵심
LM이 "정답이 검색 문서에서 어떻게 도출되는지"를 스스로 설명한 근거를 생성하고, 그것을 인컨텍스트 시연 또는 SFT 데이터로 활용해 노이즈 제거를 명시적으로 학습시키는 RAG 방법.
