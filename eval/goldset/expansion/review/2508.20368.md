# 2508.20368 AI-SearchPlanner: Modular Agentic Search via Pareto-Optimal Multi-Objective Reinforcement Learning

> section_tag: 질의계획;에이전트오케스트레이션;학습형RL  ·  survey_source: s2;s3

## 초록 (한글)
최근 연구들은 대형 언어 모델(LLM)과 검색 엔진을 통합하여 LLM의 내부 사전학습 지식과 외부 정보를 함께 활용하는 방향을 탐구해 왔다. 특히 강화학습(RL)은 검색 엔진과의 다중 턴 상호작용을 통해 LLM 추론을 향상시키는 유망한 패러다임으로 부상했다. 그러나 기존 RL 기반 검색 에이전트는 단일 LLM이 검색 계획(search planning)과 질의응답(QA)을 모두 종단간(end-to-end)으로 처리하기 때문에 두 능력을 동시에 최적화하기 어렵다. 실제로 정교한 AI 검색 시스템은 고품질 QA를 보장하기 위해 크고 고정된(frozen) LLM(예: GPT-4, DeepSeek-R1)을 사용하는 경우가 많다. 따라서 더 효과적이고 효율적인 접근은 검색 계획만을 전담하는 작고 학습 가능한 LLM을 사용하는 것이다. 본 논문은 검색 계획에 집중함으로써 고정된 QA 모델의 성능을 끌어올리도록 설계된 새로운 강화학습 프레임워크 AI-SearchPlanner를 제안한다. 핵심 혁신은 세 가지다: 1) 검색 계획기와 생성기의 아키텍처 분리, 2) 검색 계획을 위한 이중 보상(Dual-Reward) 정렬, 3) 계획 효용과 비용의 파레토 최적화. 실제 데이터셋에 대한 광범위한 실험에서 AI-SearchPlanner는 효과성과 효율성 모두에서 기존 RL 기반 검색 에이전트를 능가했으며, 다양한 고정 QA 모델과 데이터 도메인에 대해 강한 일반화 능력을 보였다.

## Introduction (한글)
LLM과 검색 엔진을 결합해 내부 지식과 외부 정보 검색(IR)을 함께 활용하려는 연구가 복잡한 추론 과제 성능을 높이는 방향으로 진행돼 왔다. 초기 대표 패러다임인 검색 증강 생성(RAG)은 입력 질의에 따라 관련 구절을 검색해 LLM 문맥에 넣어 생성한다. 다만 대부분의 RAG 시스템은 정적 검색에 의존해 적응적 계획 능력이 없고 검색 정확도에 성능이 크게 좌우된다. LLM의 일반 능력이 발전하면서, 최근 연구는 검색 엔진을 도구로 활용하는 방향을 탐구했는데, 주로 프롬프트 기반 방법이나 자기지도 미세조정(SFT)을 사용한다. 그러나 프롬프트 기반은 사전학습 분포 밖의 복잡한 추론에서 일반화가 제한적이고, SFT 기반은 대규모 고품질 검색-추론 궤적 주석이 필요해 확장성 문제가 있으며, 암기된 추론 경로에 과적합되어 분포 외(OOD) 일반화가 제한된다.

최근 RL은 LLM 추론 능력을 향상시키는 유망한 패러다임으로 부상했다. OpenAI-o1, DeepSeek-R1 같은 모델은 RL 기법(예: PPO, GRPO)으로 논리 추론과 문제 해결을 강화한다. Search-R1은 다중 턴 추론에서 QA 보상을 활용해 LLM이 검색 엔진과 상호작용하도록 훈련함으로써 RL의 잠재력을 보였다. 그러나 기존 RL 기반 검색 에이전트는 보통 단일 LLM이 검색 계획과 QA를 종단간으로 처리해 두 능력을 동시에 최적화하기 어렵다. 실제 시스템(예: Baidu, Tencent Yuanbao)은 고품질 QA를 위해 크고 고정된 LLM을 쓰므로, 검색 계획만 전담하는 작고 학습 가능한 LLM을 쓰는 것이 더 효과적·효율적이다.

이를 위해 본 논문은 검색 계획에 집중해 종단간 QA 성능을 높이는 RL 프레임워크 AI-SearchPlanner를 제안한다. 세 가지 혁신: 1) 검색 계획기/생성기 아키텍처 분리 — QA는 크고 고정된 생성기 LLM에 맡기고, 작고 학습 가능한 검색 계획기 LLM이 검색 계획에 집중. 2) 이중 보상 정렬 — 결과 수준 보상(비계획 베이스라인 대비 계획의 성능 이득 측정)과 과정 수준 보상(검색 계획 궤적의 합리성 평가). 3) 계획 효용과 비용의 파레토 최적화 — QA 정확도와 계산 비용(추론 턴 수)의 트레이드오프를 파레토 최적화 문제로 정식화. 다중 검색 추론 데이터셋 실험에서 효과성·효율성 모두 기존 RL 검색 에이전트를 능가했다.

## 한 줄 핵심
크고 고정된 QA 생성 LLM은 그대로 두고, 작고 학습 가능한 "검색 계획기" LLM만 RL로 따로 훈련해 검색 계획의 효용과 비용을 파레토 최적화하는 모듈형 에이전트 검색 프레임워크.
