# 2501.10053 AirRAG: Activating Intrinsic Reasoning for Retrieval Augmented Generation via Tree-based Search

> section_tag: 추론워크플로우  ·  survey_source: s1

## 초록 (한글)
대형언어모델(LLM)의 자율적 의사결정 능력 활용은 추론 과제에서 우수한 성능을 보여왔다. 그러나 반복적 또는 에이전트형 검색증강생성(RAG) 기법의 성공에도 불구하고, 이들은 복잡한 문제에 직면할 때 단일 해 공간에 제약되는 경우가 많다. 본 논문에서 우리는 자율적 전략 계획과 효율적 추론 행동을 통합하는 새로운 RAG 사고 패턴을 제안한다. 이는 몬테카를로 트리 탐색(MCTS)을 통해 내재적 추론 능력을 크게 활성화하고 특정 과제의 해 공간을 확장하며, AirRAG라 부른다. 구체적으로 우리 접근법은 다섯 가지 기본 추론 행동을 설계하고, 이를 MCTS로 광범위한 트리 기반 추론 공간으로 확장한다. 또한 자기일관성 검증을 통합하여 잠재적 추론 경로와 추론 스케일링 법칙을 탐색한다. 추가로 계산 최적 전략을 사용해 핵심 행동에 더 많은 추론 자원을 할당하여 전반적 성능을 높인다. 실험 결과 AirRAG의 효과가 입증되었으며, 복잡한 질의응답 데이터셋에서 상당한 성능 향상을 보였다. 또한 AirRAG는 유연하고 경량이어서 다른 고급 기술·모델과 쉽게 통합된다.

## Introduction (한글)
RAG는 특히 도메인 특화·지식집약적 과제에서 사실 오류 생성 문제 해결에 큰 잠재력을 보였다. 그러나 과제 복잡도가 증가하면 단일 질의로 충분한 지식을 검색하지 못하거나, 질문에 내재된 복잡한 추론 논리를 이해하기 어려운 새 도전이 나타난다. 이를 위해 LLM의 추론 능력을 활용해 RAG 성능을 개선하는 것이 중요하다.

복잡 질의 시나리오에 대한 이전 연구는 주로 질의·검색 과정 최적화에 집중했다. 반복적 검색은 정보 검색 과제에서 검색 결과의 깊이와 관련성을 높이기 위해 자주 쓰이며, 중간 질의·결과를 지속 갱신한다. 또한 Li et al.(2025)은 에이전트형 검색 워크플로우를 추론 과정에 통합하여 LLM이 불확실한 지식점에 직면할 때 동적 검색을 가능케 했다. 에이전트형 LLM은 강화학습을 통해 검색과 함께 단계적 추론을 학습하도록 훈련된다.

그러나 이들 접근은 두 가지 중요한 문제에 직면한다. 첫째, 단일 추론 패러다임과 사슬형 추론 과정은 해 공간을 효과적으로 탐색하지 못하며, 특히 자기탐색에 의존할 때 저품질 중간 추론 단계에 취약하고 좁은 해 공간에 갇히기 쉽다. 둘째, 에이전트형 검색 워크플로우와 자기탐색 안내는 상대적으로 작은 언어모델(예: Qwen2.5-7B-Instruct) 사용 시 어렵고, 학습 가능한 에이전트형 LLM은 효율적 RL 학습 데이터를 요구하며 수천억 파라미터 모델에 적용하기 어렵다.

이에 우리는 AirRAG를 제안한다. 자율적 전략 계획·추론 능력을 활용하고 MCTS로 해 공간을 확장한다. 다섯 가지 기본 추론 행동(시스템 분석, 직접 답변, 검색-답변, 질의 변환, 요약-답변)을 설계하며, 이들은 심층 검색 시나리오에서 핵심적이고 다양한 규모의 LLM에서 효율적으로 실행된다. 또한 MCTS와 자기일관성을 도입하여 제어 가능한 추론 경로 생성과 효율적 추론 스케일링을 가능케 하고, 다수 추론 경로에서 답을 정확히 선택하기 위해 투표 메커니즘과 과정 감독 보상 모델을 결합한다.

## 한 줄 핵심
RAG에 다섯 가지 추론 행동과 몬테카를로 트리 탐색(MCTS)·자기일관성을 결합하여, 사슬형 단일 추론의 좁은 해 공간을 트리로 넓혀 복잡 QA를 푸는 방법이다.
