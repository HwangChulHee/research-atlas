# 2604.19766 OThink-SRR1: Search, Refine and Reasoning with Reinforced Learning for Large Language Models

> section_tag: 검색시점;학습형RL  ·  survey_source: s3

## 초록 (한글)
검색증강생성(RAG)은 LLM의 지식을 확장하지만, 현재의 정적 검색 방법은 복잡한 멀티홉 문제에서 어려움을 겪는다. 최근 동적 검색 전략이 개선을 주지만 두 가지 핵심 난제에 직면한다. 1) 무관한 검색 잡음이 추론 과정을 오도할 수 있고, 2) 전체 문서를 처리하면 막대한 계산·지연 비용이 든다. 이를 해결하기 위해, 반복적 Search-Refine-Reason 과정을 강화학습으로 훈련해 대형 모델을 강화하는 프레임워크 OThink-SRR1을 제안한다. 핵심 Refine 단계는 추론 전에 검색 문서를 간결하고 관련성 있는 사실로 증류한다. 정확한 증거 식별을 보상하고 과도한 검색을 벌하는 종단간 RL 알고리즘 GRPO-IR을 도입해 모델이 집중적·효율적이 되도록 훈련한다. 네 개 멀티홉 QA 벤치마크 실험에서, 더 적은 검색 단계와 토큰을 쓰면서 강한 베이스라인 대비 우월한 정확도를 달성한다. 이로써 OThink-SRR1을 정보 탐색 에이전트의 강력한 기반 모델로 자리매김한다.

## Introduction (한글)
LLM은 대규모 사전학습으로 뛰어난 성과를 냈으나 내부 지식에만 의존하면 환각·구식 사실·지식 공백을 낳는다. RAG는 외부 검색 엔진/지식 베이스를 생성에 통합해 출력을 실세계 증거에 근거화한다. 대부분의 RAG는 단일 단계 "retrieve-reason" 또는 고정 검색 계획의 다단계 "plan-retrieve-reason" 패러다임을 따르는데, 중간 발견에 동적으로 적응하지 못해 중복 검색과 검색 문서의 비효율적 활용을 낳는다.

DeepSeek-R1, OpenAI-o1, Qwen-QwQ 같은 추론 지향 LLM의 발전은 사고연쇄 계획과 도구 호출을 크게 향상시켰다. 강한 추론 능력은 복잡 질의의 정밀 분해, 부분 증거 기반 후속 검색의 동적 조정, 다수 문서 간 종합·충돌 해소를 가능케 한다. 이 시너지로 R1-Searcher, Search-R1, ReSearch 같은 동적 추론-검색 모델/프레임워크가 등장해 추론과 검색 동작을 교차하며 더 유연한 정보 수집을 한다.

그러나 기존 동적 검색-추론 방법은 두 난제에 직면한다. 첫째, 걸러지지 않은 검색 결과는 무관 내용을 상당량 포함해 추론을 오도한다—검색의 60% 이상이 오류를 유발하고, 프롬프트가 길어질수록 추가 맥락의 효용이 떨어지며 특히 작은 모델에서 심하다. 둘째, 전체 텍스트 문서를 프롬프트에 붙이면 토큰·계산 비용이 급증해 추론 효율과 지연이 악화된다. 잡음 억제와 추론 효율 모두를 다뤄야 진정 효과적인 동적 검색 추론이 가능하다.

이를 해결하고자 OThink-SRR1을 제안한다. 검색 결과가 얻어지면 즉시 현재 맥락에 가장 관련된 핵심 정보를 추출하도록 정제하고, 원 문서는 제거해 후속 추론이 정제된 내용에만 집중하도록 한다. 또한 정답이 마지막 검색 문서·정제 요약·최종 답에서 발견될 때 보상하고 과도한 검색을 벌하는 보상 함수를 설계해, 더 정밀한 검색 질의 생성, 검색 효율 향상, 잡음 필터링을 유도한다. Qwen2.5-7B/3B-Instruct에서 훈련해 네 개 멀티홉 QA 벤치마크에서 평가하며, 더 높은 정확도와 더 나은 검색 효율, 다라운드에서 더 낮은 생성 비용을 보인다. 기여: (1) 반복적 Search-Refine-Reason 능력을 RL로 강화하는 OThink-SRR1 제안(검색 내용 정제를 종단간 학습으로 내재화·최적화한 최초의 대형 검색 추론 모델), (2) 정답 적중을 보상하고 과도 검색을 벌하는 RL 알고리즘 GRPO-IR 제안, (3) 멀티홉 QA에서 EM/F1 우월성과 평균 검색 횟수·총 토큰 사용량 감소를 체계적으로 검증.

## 한 줄 핵심
검색-정제(핵심 사실만 증류)-추론을 반복하며 GRPO-IR로 정확한 증거 적중을 보상·과도 검색을 벌해, 잡음과 비용을 동시에 줄이는 멀티홉 검색 추론 모델.
