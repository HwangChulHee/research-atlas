# 2506.08625 RAISE: Enhancing Scientific Reasoning in LLMs via Step-by-Step Retrieval

> section_tag: 질의계획  ·  survey_source: s2

## 초록 (한글)
과학적 추론은 장쇄(long-chain) 추론 과정뿐 아니라 도메인 특화 용어 지식과 갱신된 발견에 대한 적응을 요구한다. 이러한 과학적 추론의 난제를 다루기 위해, in-the-wild 코퍼스에서 논리적으로 관련된 문서를 검색하는 단계별 검색 증강 프레임워크 RAISE를 제안한다. RAISE는 (1) 문제 분해, (2) 논리적 질의 생성, (3) 논리적 검색의 세 단계로 구성된다. RAISE는 과학적 추론 벤치마크에서 다른 기준 모델들을 일관되게 능가함을 관찰한다. 분석 결과, RAISE는 다른 기준 모델과 달리 도메인 지식 측면에서 유사할 뿐 아니라 논리적으로 더 관련된 문서를 검색한다.

## Introduction (한글)
LLM은 고급 추론, 도메인 특화 용어, 최신 지식을 요구하는 과학적 추론에서 강한 잠재력을 보였다. 두 가지 흔한 전략은 (a) 구조화된 중간 단계로 복잡한 문제를 푸는 단계별 추론과, (b) 외부 증거를 제공해 환각을 완화하는 RAG다. 최근 연구는 둘을 결합하지만, 대개 더 단순한 멀티홉 QA를 대상으로 하거나 위키피디아 같은 오픈 도메인이 아닌 정제된 과제별 코퍼스를 가정한다. 대학원 수준 생물학·화학 같은 도전적 과학 추론을 in-the-wild 코퍼스로 푸는 것은 어렵다. 단순히 표면적 지식을 검색하는 것으로는 부족하며, 검색된 정보가 문제 해결에 필요한 논리적 연결을 담아야 하기 때문이다. 게다가 같은 문제 안에서도 각 중간 단계에 필요한 지식이 크게 달라질 수 있다. 각 추론 과정마다 변화하는 정보 수요를 고려하지 않으면 RAG가 오히려 다운스트림 성능을 악화시킬 수 있다. 과학 추론 과제를 풀 때 각 단계에 대해 무엇을 검색하고 어떻게 적절한 외부 지식을 검색할지는 충분히 탐구되지 않았다.

이를 해결하기 위해 단계별 과학적 추론에 맞춘 검색 증강 프레임워크 RAISE를 제안한다. 세 단계로 구성된다. (1) 문제 분해: LLM이 원 질문을 검색 질의와 함께 하위 질문으로 분해한다. (2) 논리적 질의 생성: 각 검색 질의를 하위 질문 해결에 필요한 추론을 포착하는 논리 강화 질의로 재구성한다. (3) 논리적 검색: 오픈 도메인 코퍼스에서 단계별 문서를 검색하되, 표면적 도메인 유사성이 아닌 논리적 관련성을 보장한다. 과제 관련·정제된 검색 소스를 가정하는 대신 위키피디아 같은 in-the-wild 소스에서 검색하여 어려운 실제 시나리오에 적용 가능하다. GPQA, SuperGPQA, MMLU에서 평가한 결과, RAISE는 RAG 단독 또는 문제 분해 단독 기준선을 일관되게 능가하며, 복잡한 과학 추론에 필수적인 단계별·논리적으로 관련된 정보 검색 능력을 입증한다. 본 연구는 과학적 추론에 집중하지만, 정밀한 다단계 논리 추론을 요구하는 수학적 추론에서도 유사한 난제가 발생한다.

## 한 줄 핵심
과학 문제를 하위 질문으로 분해하고 각 단계마다 "논리적으로 관련된" 문서를 검색하도록 질의를 재구성하는 3단계 단계별 RAG 프레임워크.
