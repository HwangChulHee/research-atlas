# 2508.10874 SSRL: Self-Search Reinforcement Learning

> section_tag: 검색시점  ·  survey_source: s3

## 초록 (한글)
본 논문은 강화학습(RL)에서 에이전트형 검색 과제를 위한 효율적 시뮬레이터로 대형 언어 모델(LLM)이 기능할 잠재력을 조사하여, 비용이 큰 외부 검색 엔진과의 상호작용 의존을 줄이고자 한다. 이를 위해 먼저 구조화된 프롬프팅과 반복 샘플링을 통해 LLM의 내재적 검색 능력을 정량화하며, 이를 Self-Search라 명명한다. 결과적으로 LLM은 추론 예산에 대해 강한 스케일링 거동을 보이며, 도전적인 BrowseComp 과제를 포함한 QA 벤치마크에서 높은 pass@k를 달성한다. 이 관찰을 바탕으로, 형식 기반·규칙 기반 보상으로 LLM의 Self-Search 능력을 강화하는 Self-Search RL(SSRL)을 도입한다. SSRL은 외부 도구 없이 모델이 내부적으로 지식 활용을 반복적으로 정교화하도록 한다. 실증 평가는 SSRL로 학습된 정책 모델이 검색 기반 RL 학습에 비용 효율적이고 안정적인 환경을 제공하며 외부 검색 엔진 의존을 줄이고 견고한 sim-to-real 전이를 촉진함을 보여준다. 결론: 1) LLM은 효과적으로 끌어낼 수 있는 세계 지식을 보유하며, 2) SSRL은 내부 지식을 활용해 환각을 줄일 잠재력을 보이고, 3) SSRL로 학습된 모델은 추가 노력 없이 외부 검색 엔진과 매끄럽게 통합된다.

## Introduction (한글)
최근 검증 가능한 보상을 사용한 RL은 복잡한 수학 문제 해결과 코드 생성에서 LLM의 추론 능력을 크게 향상시켜 대형 추론 모델(LRM)의 등장을 이끌었다. 수학·코딩을 넘어 많은 연구가 도구 학습 같은 에이전트 맥락에 RL을 적용해, LLM이 웹 검색 엔진 같은 외부 도구를 호출하고 행동하며 실제 환경에서 상태를 관찰하도록 학습시킨다. Search-R1, Kimi V2 같은 최근 모델이 강한 성능을 보였으나, 실제 웹 검색 엔진과의 상호작용은 여전히 비용이 크다(특히 RL 학습 중 다수의 롤아웃과 다중 턴 도구 호출 필요).

사실 웹 규모 코퍼스 사전학습 덕분에 LLM은 세계 지식 관련 질문에 자주 답할 수 있다. 일부 연구는 LLM이 주어진 행동에 대해 상태 정보를 제공하는 세계 모델로 기능할 수 있음을 시사한다. 예컨대 ZeroSearch는 미세조정된 LLM이 웹 검색을 효과적으로 대체해 안정적이고 신뢰할 만한 지식을 제공함을 보였는데, 이는 semi-real 설정 채택으로 검색 RL 비용을 크게 줄일 수 있음을 시사한다. TTRL 같은 비지도 RL의 최근 진전에 영감받아, 학습 중 실제 검색을 전혀 쓰지 않는 완전 시뮬레이션(full-sim) RL 설정에서 self-search RL을 탐구한다. 두 핵심 연구 질문: 1) 내부 지식만 쓸 때 검색 기반 QA 과제에서 LLM의 성능 한계는? 2) full-sim 검색 RL이 추론 시 실제 웹 검색과의 효과적 sim-to-real 전이를 가능케 하는가?

먼저 LLM이 매개변수에 내장된 지식을 바탕으로 질의와 정보를 모두 생성해 외부 검색 엔진 질의를 효과적으로 시뮬레이션할 수 있는지 조사한다. 모델이 다중 턴·도구 형식 출력으로 단일 생성 궤적 안에서 검색 과정을 시뮬레이션하도록 프롬프팅해 내재적 검색 능력을 평가한다. 광범위한 샘플링 결과, LLM은 상당한 세계 지식을 매개변수에 인코딩하며 스케일링 법칙을 따르는 높은 pass@k를 낸다. 다만 최적 답을 신뢰성 있게 추출하는 것은 여전히 어려워, 잠재 지식과 실행 가능한 검색 사이의 간극이 드러난다. 이 과제를 다루고 full-sim 검색 RL의 sim-to-real 잠재력을 탐구하기 위해, 형식 기반·규칙 기반 보상으로 LLM의 self-search 능력을 강화하는 SSRL을 연구한다. SSRL로 학습된 모델은 Search-R1, ZeroSearch 같은 검색 API 기반 RL 기준선을 능가할 뿐 아니라, 비용 효율적인 암묵적 세계 지식 제공자로 기능해 환각을 줄이고 외부 검색 의존을 낮추며 sim-to-real 일반화의 새 길을 연다.

## 한 줄 핵심
외부 검색 엔진 대신 LLM 자신의 내부 지식으로 검색을 시뮬레이션(Self-Search)하고, 형식·규칙 기반 보상의 RL로 이를 강화해 저비용·안정적 검색 RL 학습과 sim-to-real 전이를 가능케 하는 방법.
