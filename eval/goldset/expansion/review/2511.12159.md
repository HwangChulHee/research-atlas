# 2511.12159 CriticSearch: Fine-Grained Credit Assignment for Search Agents via a Retrospective Critic

> section_tag: 질의계획  ·  survey_source: s3

## 초록 (한글)
검색 엔진을 사용한 도구 통합 추론(Tool-Integrated Reasoning, TIR)은 대형 언어 모델이 최신 외부 지식을 반복적으로 검색하도록 하여, 복잡한 질의응답 과제에서 적응성과 일반화를 향상시킨다. 그러나 기존 검색 에이전트 파이프라인은 보통 강화학습 기반 최적화에 의존하는데, 이는 희소한 결과 보상(sparse outcome rewards)으로 인해 비효율적 탐색과 불안정한 훈련을 자주 겪는다. 우리는 회고적 비평(retrospective critic) 메커니즘을 통해 조밀한 턴 수준(turn-level) 피드백을 제공하는 세밀한 신용 할당(credit-assignment) 프레임워크 CriticSearch를 소개한다. 훈련 중, 고정된(frozen) 비대칭 비평 LLM이 전체 궤적과 정답이라는 특권 정보(privileged information)를 사용해 각 턴을 회고적으로 평가하고, 이 평가를 정책 개선을 안내하는 안정적이고 조밀한 보상으로 변환한다. 다양한 다중 홉 추론 벤치마크에서의 실험 결과, CriticSearch는 기존 베이스라인을 일관되게 능가하며 더 빠른 수렴, 향상된 훈련 안정성, 더 높은 성능을 달성했다.

## Introduction (한글)
최근 다중 턴 도구 통합 추론(TIR)은 코드 생성, 수학 추론, 질의응답 등 광범위한 다운스트림 과제에서 큰 잠재력을 보였다. 이 패러다임은 강화학습(RL)의 상호작용·반복 메커니즘에서 영감을 얻어, 모델이 외부 도구(예: 검색 엔진, 코드 인터프리터)를 호출하고 최신 정보에 접근하며 피드백 기반 반복으로 추론 과정을 점진적으로 정제하게 한다. 전통적 단일 턴 추론과 비교해 TIR은 오래된 지식·불충분한 문맥 같은 LLM의 본질적 한계를 효과적으로 완화해, 복잡한 Q&A·문제 해결에서 더 강한 성능과 일반화를 달성한다.

그러나 TIR의 효과는 RL 기반 최적화에 달려 있다. 기존 알고리즘은 보통 희소한 결과 보상에 의존하는데, 궤적 내 모든 행동이 품질과 무관하게 같은 보상을 공유한다. 이런 거친 피드백은 모델이 효과적·비효과적 도구 호출을 구별하지 못하게 해 비효율적 탐색과 느린 정책 개선으로 이어진다. 또한 보상 희소성은 리턴의 높은 분산을 유발해 훈련을 불안정하게 만든다. 이런 도전은 깊은 검색(검색 에이전트)이나 빈번한 도구 호출 설정에서 특히 심각해지며, 외부 생성 토큰이 많아 불안정을 악화시켜 보상 붕괴와 그래디언트 폭발을 초래하기도 한다.

자연스러운 해법은 희소한 전역 보상에만 의존하지 않고 세밀한 턴 수준(행동 수준) 피드백을 제공하는 것이다. 그러나 기존 검색 에이전트 패러다임은 희소 보상 하에서 효과적 신용 할당을 수행하기 어렵다(표 1). 일부 연구는 몬테카를로로 세밀한 보상을 근사하나, 분산이 높고 안정적 추정에 대량 롤아웃이 필요하다. 일부는 정답 참조로 지도하려 하나, 주석 데이터에 크게 의존하고 일반성이 부족하다.

이를 극복하기 위해 우리는 CriticSearch를 제안한다 — LLM을 활용해 정책 훈련을 돕는 세밀한 신용 할당 방법. 핵심 아이디어는 회고적 비평 메커니즘 도입이다: 모델이 완전한 추론 궤적을 생성하고 정답을 얻은 후, 비평 모델이 각 중간 행동을 다시 보며 최종 결과에 대한 기여를 평가해 세밀한 턴 수준 보상을 생성한다. 전향적 의사결정과 달리 이 회고적 평가는 훈련 중에만 비대칭적으로 가용한 특권 정보(예: 정답, 미래 상호작용 턴)를 활용한다. 이런 후견(hindsight) 접근은 턴 수준 보상을 더 쉽고 신뢰성 있게 생성한다. 턴 수준과 전역 보상을 통합해, CriticSearch는 중복·저품질 행동에 낮은 보상을, 핵심 행동의 학습 신호는 증폭한다. 기여: 회고적 비평(미세조정 없이 사전학습 LLM으로 조밀 보상 생성), 조밀 보상 도입의 핵심 현상(가속 수렴·불안정 완화) 분석, 3B·7B 모델에서 조밀 보상 베이스라인 대비 각각 16.7%·6.7% 향상.

## 한 줄 핵심
검색 에이전트 RL의 희소 보상 문제를, 정답과 전체 궤적을 본 고정 비평 LLM이 각 턴을 회고적으로 채점해 조밀한 턴 수준 보상을 주는 세밀한 신용 할당 방법.
