# 2602.00994 Reasoning and Tool-use Compete in Agentic RL: From Quantifying Interference to Disentangled Tuning

> section_tag: RAG강화추론  ·  survey_source: s3

## 초록 (한글)
에이전트형 강화학습(ARL)은 LLM이 복잡한 과제를 풀기 위해 추론과 외부 도구 실행을 번갈아 수행하도록 훈련한다. 기존 ARL 방법 대부분은 추론과 도구 사용 행동을 모두 지원하도록 단일 파라미터 집합을 훈련하며, 공동 훈련이 전반적 에이전트 성능 향상으로 이어진다고 암묵적으로 가정한다. 널리 채택되었음에도 이 가정은 경험적으로 거의 검증된 바 없다. 본 논문에서 우리는 추론과 도구 사용 행동 간 간섭의 정량적 증거를 제공하는 능력 효과 귀속(Capability Effect Attribution, CEA)을 도입해 이 가정을 체계적으로 검토한다. 심층 분석을 통해 두 능력이 종종 어긋난 그래디언트 방향을 유발하여 공동 최적화의 효과를 저해하는 훈련 간섭을 일으키며, 이는 기존 ARL 패러다임에 도전한다는 것을 보인다. 이를 해결하기 위해 우리는 별도의 저랭크 적응(LoRA) 모듈로 추론과 도구 사용의 파라미터 갱신을 명시적으로 분리하는 단순하고 효율적인 프레임워크 DART(Disentangled Action–Reasoning Tuning)를 제안한다. 이 단순한 변경만으로 DART는 모든 공동 최적화 기준선을 능가하고, 검색증강 QA와 NL2SQL의 13개 벤치마크에서 2-에이전트 상한선에 근접하여 공유 최적화 하의 능력 간섭 발견을 뒷받침한다.

## Introduction (한글)
사후 훈련을 위한 에이전트형 강화학습(ARL)의 최근 발전은 LLM의 능력을 크게 확장했다. 현대 LLM은 텍스트 생성을 넘어 복잡한 추론을 수행하고 외부 도구와 상호작용하여 정보 검색, 계산, 데이터 분석, 연구 워크플로우 같은 과제를 푼다.

ARL의 목표는 강한 추론 능력을 보이면서 외부 도구를 안정적으로 실행하는 모델을 훈련하는 것이다. 기존 ARL 패러다임 대부분은 단일 ARL 목표와 공유 파라미터로 이 두 이질적 능력을 공동 최적화한다. 이 설계는 도구 실행과 논리적 추론이 같은 파라미터 공간에 효과적으로 수용될 수 있다고 암묵 가정한다. 선행 연구는 서로 다른 도메인 간 훈련에서 최적화 간섭이 생김을 보였으나, 단일 에이전트 도메인 내 이질적 능력들도 서로 간섭하는지는 거의 탐구되지 않았다.

본 연구에서 우리는 도구 사용과 추론 간 상호작용의 통제된 경험 분석을 통해 공유 파라미터 가정을 직접 검증한다. 구체적으로 에이전트 성능을 개별 능력 효과와 쌍별 상호작용 항으로 분해하는 진단 프레임워크 CEA를 도입한다. 그래디언트 마스킹과 하이브리드 추론으로 여섯 개의 통제된 모델 변형을 구성해 질문별 상호작용 계수를 풀고, 공동 최적화 하에서 추론과 도구 사용 간 유의미한 음의 상호작용을 드러내어 공유 파라미터 훈련이 암묵적 경쟁을 유발함을 밝힌다.

이 간섭의 근본 원인을 설명하기 위해 추론과 도구 사용 토큰의 그래디언트를 분석한다. 두 유형의 그래디언트가 거의 직교하여, 공유 백본에 대한 공동 최적화가 두 능력 모두에 최적이 아닌 타협 방향으로 파라미터를 갱신함을 확인한다.

이 발견에 동기를 받아 우리는 단순하지만 효과적인 프레임워크 DART를 제안한다. DART는 사전 훈련된 백본을 동결하고 추론·도구 사용 토큰을 별도의 LoRA 어댑터로 라우팅하여 각 능력이 다른 능력에 영향을 주지 않고 자신의 파라미터만 갱신하게 한다.

7개 QA와 6개 NL2SQL 벤치마크 실험에서 DART는 공동 최적화 기준선(멀티-LoRA 포함)을 일관되게 능가하며, 토큰 수준 하드 라우팅으로 그래디언트 어긋남을 해소하는 것이 필수임을 보인다. 절제 연구는 DART의 이득이 추가 파라미터 용량이 아닌 어긋남 제거에서 비롯됨을 귀속한다.

## 한 줄 핵심
에이전트 RL에서 추론과 도구 사용이 그래디언트 차원에서 서로 간섭(거의 직교)함을 정량화하고, 두 능력을 별도 LoRA 어댑터로 분리(DART)해 간섭을 없애 성능을 올린 논문.
