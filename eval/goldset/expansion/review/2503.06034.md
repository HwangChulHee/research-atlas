# 2503.06034 Rank-R1: Enhancing Reasoning in LLM-based Document Rerankers via Reinforcement Learning

> section_tag: 정보필터링  ·  survey_source: s2

## 초록 (한글)
본 논문에서 우리는 Rank-R1을 소개한다. 이는 순위 매기기 과제를 수행하기 전에 사용자 질의와 후보 문서 양쪽에 대해 추론하는 새로운 LLM 기반 재순위기다. 기존 LLM 기반 문서 재순위 방법은 보통 LLM에게 질의 관련성에 따라 후보 문서를 정렬·라벨링하도록 프롬프팅하거나 파인튜닝하는 데 의존한다. Rank-R1의 경우, 우리는 강화학습 알고리즘과 소량의 관련성 라벨만(추론 감독 없이) 사용해 LLM 기반 재순위기의 추론 능력을 강화한다. 우리의 가설은 재순위기에 추론 능력을 더하면 관련성 평가와 순위 능력이 향상된다는 것이다. TREC DL과 BRIGHT 데이터셋 실험은 Rank-R1이 특히 복잡한 질의에서 매우 효과적임을 보여준다. 특히 인도메인 데이터셋에서 지도 파인튜닝 방법과 대등한 효과를 달성하면서도 파인튜닝 방법이 쓴 학습 데이터의 18%만 사용한다. 또한 복잡 질의가 포함된 아웃오브도메인 데이터셋에서는 제로샷·지도 파인튜닝을 크게 능가하며, 특히 14B 모델에서 두드러진다. 마지막으로 Rank-R1의 추론 과정이 순위 결과의 설명가능성을 높여 검색 엔진 결과 제시에 새로운 기회를 연다는 것을 정성적으로 관찰한다.

## Introduction (한글)
LLM은 문서 순위 과제에서 강력한 성능을 보였다. RankGPT 같은 제로샷 프롬프팅 방법은 LLM에게 직접 재정렬된 문서 목록을 생성하도록 프롬프팅하여 문서를 재순위한다. 그러나 이들은 LLM이 지시를 잘 따른다는 가정에 의존하고, 제로샷이라 가용한 인간 주석 관련성 데이터를 추가 개선에 활용하지 못한다. 반면 RankLlama 같은 방법은 인간 관련성 판단으로 LLM을 파인튜닝해 질의-문서 쌍에 점수를 매긴다. 효과적이지만 이들은 추론 과정을 명시적으로 모델링하지 않는데, 이는 주로 지도 파인튜닝용 고품질 추론 데이터의 부재 때문이다. 실제로 사용자 관련성 판단은 종종 집합에서 가장 관련된 문서를 선택하는 형태로 오지만, 고품질 순위에는 복잡한 관련성 관계를 해석하는 강한 추론 능력이 필요하다.

DeepSeek-R1, Simple RL 같은 LLM용 강화학습(RL)의 최근 발전은 보상 기반 훈련이 특히 수학 질의응답 같은 과제에서 추론 능력을 강화할 수 있음을 보였다. 이러한 혁신은 규칙 기반 보상 점수만으로도 LLM의 추론·설명 능력을 향상시킬 수 있음을 시사한다.

이 통찰에 기반하여, 우리는 DeepSeek-R1 스타일의 강화학습이 문서 재순위의 추론을 개선할 수 있는지 묻는다. 구체적으로 우리는 Group Relative Policy Optimization(GRPO)을 적용해 LLM 기반 재순위기를 훈련한다. 사용자 질의와 검색된 후보 문서 목록이 주어지면, 재순위기는 가장 관련된 문서를 선택하기 전에 추론 단계를 생성한다. 보상 신호는 모델이 결국 후보 중 가장 관련된 문서를 정확히 식별하는지에 의해서만 결정된다. 우리는 MS MARCO 패시지 순위 데이터셋으로 Rank-R1을 훈련하고 TREC DL19/DL20(인도메인)과 BRIGHT(아웃오브도메인)에서 평가하며, 14B 모델은 훨씬 큰 (제로샷) GPT-4를 BRIGHT 재순위 성능에서 능가한다.

## 한 줄 핵심
DeepSeek-R1 스타일의 강화학습(GRPO)과 소량 관련성 라벨만으로 LLM 재순위기가 문서를 고르기 전에 추론하도록 훈련하여, 복잡 질의 재순위 성능과 설명가능성을 높인 방법이다.
